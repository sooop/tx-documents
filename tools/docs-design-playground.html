배송비 충전하기

SmartShip에서 배송비를 결제하려면 TxMoney를 먼저 충전해야 합니다. 자세한 내용은 TxMoney 안내 페이지를 참고하세요.

TxMoney란 무엇인가요?
스마트십 사용 시 발생하는 배송비와 기타 비용을 결제하기 위한 화폐입니다. 센터에 화물이 입고될 때 배송비가 자동으로 TxMoney 계좌에서 차감됩니다.

충전 방법

원하시는 충전 금액을 선택한 후, 아래 결제 방식 중 하나를 선택해주세요.

  1. 나의 TxMoney 계좌 — 현재 보유 잔액과 미결제 청구 금액을 확인할 수 있습니다.
  2. 충전 금액 입력 — 원하는 금액을 직접 입력하거나, 버튼을 눌러 빠르게 금액을 추가할 수 있습니다.
  3. 무통장입금 충전 — 가상 계좌를 발급받아 입금하는 방식입니다.
  4. Paynow / 신용·직불 카드결제 — 카드로 즉시 결제하는 방식입니다.

카드 결제 절차

  • [모든 카드 결제] 버튼을 클릭합니다.
  • 카드번호, 유효기간, CVC 코드를 입력합니다.
  • 카드 소유자 이름을 입력 후 [결제] 버튼을 클릭합니다.
무통장입금 (가상 계좌)

[가상 계좌로 입금] 버튼을 클릭합니다. 표기된 가상 계좌 번호로 입금하면 자동으로 충전이 완료됩니다.

카드 결제 유의 사항
[모든 카드 결제]는 싱가포르의 Stripe PG를 사용하고 있으며, 카드사 정책에 따라 해외 거래 수수료가 부과될 수 있습니다.

충전 및 사용 안내

  • 센터에 화물이 도착하면 배송비가 자동으로 계산되어 TxMoney 잔고에서 차감됩니다.
  • TxMoney를 충전한 후 일부 금액을 사용한 후에는 환불이 불가합니다.
  • 스마트십 서비스 탈퇴 시에는 보유 잔고의 5% 공제 후 환불 받을 수 있습니다.
팁
입고 시 TxMoney 잔고가 부족하면 입고에 실패하며 처리가 지연될 수 있습니다. 충분한 잔고를 유지해주세요.

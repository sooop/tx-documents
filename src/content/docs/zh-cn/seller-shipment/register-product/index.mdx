在 SmartShip 中注册订单之前，需要先注册待发货的商品。系统支持**单件注册**和 **Excel 批量上传**两种方式。

<Callout type="info" title="商品注册入口">
在顶部菜单进入**商品管理**，点击 **+ 注册** 按钮即可打开单件注册页面。批量注册请使用同一页面的 **Excel 上传** 按钮。
</Callout>

## 单件商品注册

商品注册页面由**商品信息**、**海外配送清关信息**、**销售渠道商品信息**三个标签页组成。按顺序填写后点击**注册**按钮即可。

### 第一步：商品信息

在**商品信息**标签页输入商品基本信息。

<AnnotatedImage src="product-reg-basic-info" section="seller-shipment" alt="商品信息标签页 — 基本信息输入页面" caption="商品信息标签页 — 商品基本信息" />

| 字段 | 必填 | 说明 |
|------|:----:|------|
| TX 商品编码 | — | 系统自动分配，不可修改 |
| 卖家自定义编码 | | 卖家自行设定的内部管理编码 |
| 商品条形码 | | 输入条形码后点击**确认**，可自动调取已有商品信息 |
| 商品名称 | ✓ | 至少输入 3 个字符 |
| 选项名称 | | 如有颜色、尺寸等选项，在此输入 |
| 价格 / 货币 | | 销售单价及货币单位（默认：KRW） |
| 重量（kg） | | 输入 0.1 以上的数值 |
| 尺寸（cm × cm × cm） | | 宽 × 长 × 高 |
| 原产地 | | 从下拉菜单中选择 |
| 材质 | | 选择材质类型或手动输入 |
| 含电池 | | 选择是否含电池（默认：不含） |
| 敏感商品 | | 选择适用 / 不适用 |
| HS 编码 | | 手动输入关税分类编码 |

### 第二步：海外配送清关信息

在**海外配送清关信息**标签页填写出口清关所需信息。

<AnnotatedImage src="product-reg-customs-info" section="seller-shipment" alt="海外配送清关信息标签页输入页面" caption="海外配送清关信息标签页 — 商品清关信息管理" />

| 字段 | 说明 |
|------|------|
| 清关商品名称 | 出口申报时使用的英文品名，必须以**英文大写**输入 |
| 商品条形码 | 条形码编号 |
| 销售网站 URL | 商品所在销售页面的 URL |
| HS / HSN | 按大类 → 中类 → 小类依次选择，或直接输入编码 |
| 材质 | 材质种类及成分比例 |
| 含电池 | 是否含电池 |
| 敏感商品 | 是否属于敏感品目 |
| 型号 / 品牌 | 产品型号及品牌名称 |
| 原产地 / 制造商 | 生产国及制造商名称 |
| 清关商品名称（zh-cn） | 发往中国时填写中文品名 |
| 商品税号 | 海关申报用税则号 |
| GST 税率（%） | 发往 GST 适用国家（如澳大利亚）时填写（默认：0） |
| VAT ID | VAT 登记编号 |
| EORI | 发往欧洲时填写 EORI 编号 |

<Callout type="warning" title="清关商品名称注意事项">
清关商品名称必须以**英文大写**填写。使用中文、韩文或特殊符号可能导致清关错误。
</Callout>

### 第三步：销售渠道商品信息

在**销售渠道商品信息**标签页中，可查看并配置各已对接销售平台的商品信息。

填写完所有信息后，点击**注册**按钮即可完成商品注册。

## 批量商品注册（Excel 上传）

需要一次性注册多件商品时，请使用 Excel 上传功能。

<AnnotatedImage src="product-bulk-reg" section="seller-shipment" alt="批量配送商品注册 — Excel 上传页面" caption="批量配送商品注册页面" />

### 下载 Excel 模板

1. 点击批量配送商品注册页面右上角的 **下载 XLSX 格式** 按钮。
2. 打开下载的 Excel 文件，填写商品信息。

<Callout type="warning" title="新版模板说明（自 2024 年 7 月 11 日起适用）">
新版模板的列名中包含编码和括号，请**勿删除**这些编码及括号，否则上传将无法正常处理。
</Callout>

### 列说明

<Collapse title="查看全部列说明">

| 列名 | 编码 | 说明 |
|------|------|------|
| 商品名称 | M00 | 商品名称（必填） |
| 发货数量 | M01 | 每次发货的数量 |
| 单价 | M02 | 商品单价 |
| 货币 | M03 | 货币代码（如：KRW、USD、JPY） |
| 细分类别 | M04 | 商品细分类别 |
| 选项 | M05 | 选项信息（颜色、尺寸等） |
| 商品 URL | M06 | 销售页面 URL |
| 商品材质（成分） | M07 | 材质及成分信息 |
| HS/JAL | M08 | HS 编码或 JAL 编码 |
| Overseas / Trade code | — | 海外贸易编码（HS Code、JAL 等） |
| 品牌 | M10 | 品牌名称 |
| Shop Name | M11 | 销售网站名称 |
| 条形码 | M12 | 商品条形码 |
| 行邮税 | M13 | 是否适用行邮税 |
| 行邮税编码（后半部分） | M14 | 行邮税编码末尾部分 |
| 卖家自定义商品编码 | M15 | 卖家内部管理编码 |
| GST Rate | M16 | GST 税率（%） |

</Collapse>

### 上传文件

1. 在下拉菜单中选择**注册**。
2. 点击**选择文件**按钮，选择已填写好的 Excel 文件。
3. 上传后，通过**处理结果（ST0）**和**详细信息（ST1）**列确认注册结果。

<Callout type="tip" title="上传失败时">
若处理结果（ST0）显示错误，请查看详细信息（ST1）了解原因，修正相应条目后重新上传。
</Callout>

## 查看和管理已注册商品

商品注册完成后，在**商品管理**菜单的商品查询中查看列表。

<AnnotatedImage src="product-management" section="seller-shipment" alt="商品管理 — 商品查询页面" caption="商品管理 — 商品查询列表" />

- **关键字搜索**：按商品编码、商品名称等条件筛选
- **下载查询内容**：将当前查询结果导出为 Excel 文件
- **Excel 上传**：用于批量修改已注册的商品信息

<Callout type="tip" title="批量修改商品信息">
如需修改已注册商品的信息，请点击 **Excel 上传**，在下拉菜单中选择**修改**，点击**下载修改格式**按钮获取当前商品数据，编辑完成后重新上传。
</Callout>

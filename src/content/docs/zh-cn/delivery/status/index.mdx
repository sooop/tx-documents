title: "配送状态"
description: "了解如何在 SmartShip 中查看已发货订单的配送状态，以及查询追踪信息和运费明细。"
---

在配送状态菜单中，您可以一目了然地查看所有已发货订单的配送状态，并逐条查询追踪信息和运费明细。

<Callout type="info" title="菜单路径">
**配送管理 → 配送状态**
</Callout>

## 查询配送订单

<AnnotatedImage src="delivery-status-page" section="delivery" alt="配送状态完整页面" caption="配送管理 > 配送状态页面" />

### 设置搜索条件

<AnnotatedImage src="delivery-status-search" section="delivery" alt="配送订单搜索筛选条件" caption="配送订单查询 — 搜索条件" />

| 搜索条件 | 说明 |
|----------|------|
| 基准日期 | 选择订单登记日、出库日等日期基准后指定查询区间（可使用 1W / 1M 按钮快速选择） |
| 配送订单类型 | 按全部订单或特定订单类型筛选 |
| 出货仓库 | 按出货仓库筛选 |
| 配送区间 | 选择出发国 → 到达国 |
| 销售站点 | 按已连接的销售渠道筛选 |
| 订单登记方式 | 按全部订单或登记方式筛选 |
| 其他条件 | 单独筛选预发号后运单号已变更的订单，或超过 150 美元的订单 |
| 关键词搜索 | 按 QSP NO.、订单号、收件人姓名等搜索 |

### 发货状态汇总

<AnnotatedImage src="delivery-status-summary" section="delivery" alt="发货状态汇总面板" caption="发货状态 — 各状态订单数量汇总" />

页面顶部的汇总面板可让您一目了然地查看各状态的订单数量。

| 状态 | 说明 |
|------|------|
| 全部配送开始 | 已开始配送的总订单数 |
| 配送中 | 当前正在配送中的订单数 |
| 配送完成 | 最终投递已完成的订单数 |
| 配送失败 | 已处理为投递失败的订单数 |
| 退件中 | 正在退件中的订单数 |
| 退件完成 | 退件已完成的订单数 |

点击任意状态项，即可直接筛选该状态的订单列表。点击右上角的**查看每日发货件数**按钮，可查看按日期统计的发货数据。

### 订单列表

<AnnotatedImage src="delivery-status-list" section="delivery" alt="配送状态订单列表" caption="配送状态 — 订单列表" />

在查询结果列表中，可查看 QSP NO.、运单号、配送状态、出发国/到达国等主要信息。点击列表底部的 **Excel** 按钮，可将查询内容下载为 Excel 文件。

## 查看订单详细信息

点击列表中的订单，底部将展开详细信息面板。面板由**配送追踪信息**、**运费信息**、**商品信息**三个标签页组成。

### 配送追踪信息

<AnnotatedImage src="delivery-status-tracking" section="delivery" alt="配送追踪信息标签" caption="配送追踪信息 — 各环节处理记录" />

可按时间顺序查看从接收配送订单到最终投递的各环节处理记录。

| 列名 | 说明 |
|------|------|
| 状态 | 处理阶段名称（配送订单接收、集货中心入库完成、出库完成、到达国清关开始/完成、移交配送商等） |
| 日期 | 状态变更的日期和时间 |
| 说明 | 附加备注（如预计出发机场等） |
| 配送员 | 当地配送员信息（如有） |

点击**配送详细信息**按钮，可跳转至配送商的原始追踪页面。

### 运费信息

<AnnotatedImage src="delivery-status-fee" section="delivery" alt="运费信息标签" caption="运费信息 — 各项费用明细" />

可查看该订单收取的运费各项明细。

| 列名 | 说明 |
|------|------|
| 运费项目 | 项目名称（基本运费、燃油附加费、EDI 传输费等） |
| 原始货币 | 计费货币单位 |
| 原始货币金额 | 各项收费金额 |

列表最底部的 **Total Fee** 行显示运费总计。

---

배송 현황 메뉴에서는 발송된 주문의 배송 상태를 한눈에 확인하고, 건별로 추적 정보와 배송비 내역을 조회할 수 있습니다.

<Callout type="info" title="메뉴 경로">
**배송관리 → 배송 현황**
</Callout>

## 배송 주문 조회하기

<AnnotatedImage src="delivery-status-page" section="delivery" alt="배송 현황 전체 페이지" caption="배송관리 > 배송 현황 화면" />

### 검색 조건 설정

<AnnotatedImage src="delivery-status-search" section="delivery" alt="배송 주문 내역 검색 필터" caption="배송 주문 내역 조회하기 — 검색 조건" />

| 검색 조건 | 설명 |
|-----------|------|
| 기준일 선택 | 주문 등록일, 출고일 등 날짜 기준 선택 후 조회 기간 지정 (1W / 1M 버튼으로 빠른 선택) |
| 배송 주문 타입 | 전체 주문 또는 특정 주문 유형으로 필터링 |
| 출하창고 | 출고 창고 기준으로 필터링 |
| 배송 구간 | 출발 국가 → 도착 국가 선택 |
| 판매 사이트 | 연동된 판매 채널별 필터링 |
| 주문 등록 방식 | 전체 주문 또는 등록 방식별 필터링 |
| 기타 조건 | 사전발번 이후 송장번호가 변경된 주문, 150불 초과 건 별도 필터링 |
| 키워드 검색 | QSP NO., 주문번호, 수취인명 등 키워드로 검색 |

### 발송 현황 요약

<AnnotatedImage src="delivery-status-summary" section="delivery" alt="발송 현황 요약 패널" caption="발송 현황 — 상태별 건수 요약" />

페이지 상단의 발송 현황 패널에서 상태별 건수를 한눈에 확인할 수 있습니다.

| 상태 | 설명 |
|------|------|
| 전체 배송 시작 | 배송이 시작된 전체 건수 |
| 배송 중 | 현재 배송이 진행 중인 건수 |
| 배송 완료 | 최종 배달이 완료된 건수 |
| 배송 실패 | 배달 실패로 처리된 건수 |
| 반송 중 | 반송이 진행 중인 건수 |
| 반송 완료 | 반송이 완료된 건수 |

각 상태 항목을 클릭하면 해당 상태의 주문 목록으로 바로 필터링됩니다. 우측 상단의 **일별 발송 건 수 확인하기** 버튼을 클릭하면 날짜별 발송 통계를 확인할 수 있습니다.

### 주문 목록

<AnnotatedImage src="delivery-status-list" section="delivery" alt="배송 현황 주문 목록 테이블" caption="배송 현황 — 주문 목록" />

조회된 주문 목록에서 QSP NO., 송장번호, 배송 상태, 출발·도착국 등 주요 정보를 확인할 수 있습니다. 목록 하단의 **Excel** 버튼을 클릭하면 조회된 내역을 엑셀 파일로 다운로드할 수 있습니다.

---

## 건별 상세 정보 확인

목록에서 주문 건을 클릭하면 하단에 상세 정보 패널이 열립니다. **배송 추적 정보**, **배송비 정보**, **상품 정보** 세 가지 탭으로 구성되어 있습니다.

### 배송 추적 정보

<AnnotatedImage src="delivery-status-tracking" section="delivery" alt="배송 추적 정보 탭" caption="배송 추적 정보 — 단계별 처리 이력" />

배송 주문 접수부터 최종 배달까지 단계별 처리 이력을 시간순으로 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| 상태 | 처리 단계명 (배송 주문 접수, 집하센터 입고완료, 출고완료, 도착국 통관 시작/완료, 배송사 인계 등) |
| 날짜 | 해당 상태로 변경된 일시 |
| 설명 | 추가 메모 (출발 예정 공항 등) |
| 배송기사 | 현지 배송기사 정보 (해당하는 경우) |

**배송 상세 정보** 버튼을 클릭하면 배송사 원본 추적 페이지로 이동합니다.

### 배송비 정보

<AnnotatedImage src="delivery-status-fee" section="delivery" alt="배송비 정보 탭" caption="배송비 정보 — 항목별 비용 내역" />

해당 주문에 청구된 배송비 항목별 내역을 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| 배송비 항목 | 기본 배송비, 유류할증료, EDI 전송료 등 항목명 |
| 원통화 | 청구 통화 단위 |
| 원통화금액 | 항목별 청구 금액 |

목록 최하단 **Total Fee** 행에서 총 배송비 합계를 확인할 수 있습니다.

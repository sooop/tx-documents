SmartShip은 해외 배송을 간편하게 처리할 수 있는 물류 플랫폼입니다.

## 주요 기능

- **다국가 배송 지원**: 일본, 미국, 싱가포르, 대만, 중국, 홍콩 등 다양한 국가로 배송
- **API 연동**: 주요 쇼핑몰과의 자동 주문 연동
- **풀필먼트 서비스**: 상품 입고부터 배송까지 원스톱 처리
- **실시간 배송 추적**: 배송 현황을 실시간으로 확인

## 이용 흐름

1. 회원가입 및 사업자 인증
2. 배송비 충전 (TxMoney)
3. 주문 등록 (직접 입력 또는 API 연동)
4. 상품 포장 및 발송
5. 배송 추적 및 관리

<Callout type="info" title="참고">
  SmartShip은 B2B 물류 서비스로, 사업자 회원만 이용 가능합니다.
</Callout>

## 지원 국가

현재 SmartShip을 통해 배송 가능한 주요 국가는 다음과 같습니다:

| 국가 | 배송 방식 | 예상 소요일 |
|------|-----------|-------------|
| 일본 | 특송 | 2-4일 |
| 미국 | 특송/일반 | 5-10일 |
| 싱가포르 | 특송 | 3-5일 |
| 대만 | 특송 | 3-5일 |

<Collapse title="더 많은 국가 확인하기">
  중국, 홍콩, 태국, 베트남, 말레이시아 등 추가 국가도 지원됩니다.
  자세한 내용은 국가별 유의사항 페이지를 참고하세요.
</Collapse>

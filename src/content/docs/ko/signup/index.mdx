SmartShip 서비스를 이용하기 위해서는 **Business 멤버** 가입 및 관리자 승인이 필요합니다.

## 회원 유형

SmartShip(TracX Logis)의 회원은 두 가지 유형으로 구분됩니다.

| 구분 | Basic 멤버 | Business 멤버 |
|------|-----------|--------------|
| 가입 조건 | 없음 (누구나) | 사업자 |
| SmartPost (개인 국내외 택배) | ✓ | ✓ |
| 배송 상황 추적 | ✓ | ✓ |
| SmartShip 시스템 이용 | — | ✓ |
| B2C 국내/해외 택배 서비스 | — | ✓ |
| B2B 화물 서비스 | — | ✓ |
| 3PL 풀필먼트 서비스 | — | ✓ |

---

## 베이직 멤버 가입 방법

Basic 멤버는 국내외 개인화물 택배 서비스인 **SmartPost** 전용 사용자입니다.

### 1단계: 가입 페이지 접속

[https://smartpost.tracxlogis.com](https://smartpost.tracxlogis.com) 으로 접속한 뒤, 화면 하단의 **[Sign Up]** 버튼을 클릭합니다.

<AnnotatedImage src="smartpost-home-signup" section="signup" alt="SmartPost 홈 화면 — Sign Up 버튼" caption="SmartPost 홈 화면 — 하단의 Sign Up 버튼 클릭" />

### 2단계: 기본 정보 입력

이메일 주소, 비밀번호, 이름, 핸드폰 번호, 국가 등 기본 정보를 입력합니다.

<AnnotatedImage src="smartpost-register-basic-01" section="signup" alt="SmartPost 기본 정보 입력 화면" caption="기본 정보 입력 — 이메일 중복 확인 필수" />

<Callout type="warning" title="이메일 중복 확인 필수">
  이메일 주소 입력 후 **[Check Duplicate]** 버튼을 반드시 클릭하세요. 중복 확인을 하지 않으면 다음 단계로 진행되지 않습니다.
</Callout>

### 3단계: 배송 주소 입력

택배 서비스 이용을 위한 수령 주소와 영문 주소를 입력합니다.

<AnnotatedImage src="smartpost-register-basic-02" section="signup" alt="SmartPost 배송 주소 입력 화면" caption="Shipping Address — 국문/영문 주소 모두 입력" />

### 4단계: 이용약관 동의 및 가입 완료

이용약관 항목에 동의한 뒤 **[Register]** 버튼을 클릭합니다.

<AnnotatedImage src="smartpost-register-basic-03" section="signup" alt="SmartPost 이용약관 동의 화면" caption="이용약관 동의 후 Register 클릭" />

## Business 멤버 가입 방법

SmartShip 시스템을 이용하려면 **사업자 회원(Business 멤버)**으로 가입해야 합니다. 가입 후 관리자 승인이 완료되어야 서비스를 이용할 수 있습니다.

### 1단계: 가입 페이지 접속

[https://smartship2.tracxlogis.com/account/register](https://smartship2.tracxlogis.com/account/register) 로 접속합니다.

### 2단계: 기본 정보 입력

이메일 주소, 비밀번호, 핸드폰 번호, 국가, TxMoney 통화를 입력합니다.

<AnnotatedImage src="smartship-register-01" section="signup" alt="SmartShip 기본 정보 입력 화면" caption="기본 정보 입력 — 이메일 중복 확인 및 TxMoney 통화 선택" />

<Callout type="warning" title="TxMoney 통화는 가입 후 변경 불가">
  TxMoney(선불충전통화)는 가입 이후 변경할 수 없습니다. 사업을 운영하는 국가에 맞는 통화를 신중하게 선택하세요.
</Callout>

### 3단계: 사업자 정보 입력 및 사업자등록증 업로드

사업자 정보를 입력하고 사업자등록증 파일을 업로드합니다. 로그인 시 이메일 대신 사용할 별도의 **로그인 ID**를 설정하려면 해당 항목에 입력 후 중복 확인을 해주세요.

<AnnotatedImage src="smartship-register-02" section="signup" alt="SmartShip 사업자 정보 입력 및 사업자등록증 업로드 화면" caption="사업자 정보 입력 — 사업자등록증 업로드 필수" />

### 4단계: 이용약관 동의 및 회원 가입 클릭

이용약관 항목에 동의한 뒤 **[회원 가입]** 버튼을 클릭합니다.

<AnnotatedImage src="smartship-register-03" section="signup" alt="SmartShip 이용약관 동의 화면" caption="이용약관 동의 후 회원 가입 버튼 클릭" />

### 5단계: 승인 요청 정보 입력

가입 신청이 완료되면 **승인 정보 입력 화면**으로 자동 전환됩니다. 판매 품목, 월 배송 건수 등 정보를 입력한 뒤 **[승인요청하기]** 버튼을 클릭하세요.

---

## 가입 완료 후 필수 사항

<Callout type="danger" title="이메일 인증 필수">
  가입 시 입력한 이메일로 **인증 메일**이 발송됩니다. 이메일 인증이 완료되지 않으면 로그인 및 SmartShip 이용에 제한이 생길 수 있습니다. 반드시 메일함을 확인하여 인증을 완료해 주세요.
</Callout>

---

## 승인은 언제 되나요?

가입 승인은 **영업일 기준 1~2일** 소요됩니다. 승인 완료 시 등록하신 이메일로 승인 안내 메일이 발송되며, 이후 SmartShip에 로그인하여 서비스를 이용할 수 있습니다.

승인이 지연될 경우 아래 연락처로 문의해 주세요.

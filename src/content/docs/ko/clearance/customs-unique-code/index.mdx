수출신고 시 필수로 기재해야 하는 **사업자 통관고유부호**를 관세청 유니패스(UNI-PASS)에서 발급받는 방법을 안내합니다.

<Callout type="info" title="통관고유부호란?">
통관고유부호는 관세청 유니패스(UNI-PASS)에 등록된 사업자를 식별하는 고유 코드입니다. 정식 수출신고(수출목록변환신고) 신청 시 반드시 입력해야 합니다.

**형식:** 상호 앞 4글자 + 업체유형 1자리 + 설립년도 2자리 + 동일업체구분 1자리 + 본지사여부 2자리 + 검증부호 1자리 (총 11자리)
</Callout>

## 유니패스 사용자 등록

[유니패스(UNI-PASS)](https://unipass.customs.go.kr)에 접속하여 업체 사용자로 등록합니다.

### 1단계: 업체 및 대표자로 등록 시작

유니패스 메인 화면 우측 상단의 **회원가입** 버튼을 클릭합니다.

<AnnotatedImage src="unipass-main-page" section="clearance" alt="유니패스 메인 화면" caption="유니패스(UNI-PASS) 메인 화면 — 우측 상단 회원가입 선택" />

### 2단계: 사용자 유형 선택

**업체 및 대표자** 항목을 선택합니다.

<AnnotatedImage src="unipass-entity-type-select" section="clearance" alt="유니패스 사용자 유형 선택" caption="업체 및 대표자 선택" />

<Callout type="info">
UNI-PASS에 업체로 등록하면, 대표자가 사업자에 연결된 사용자로 자동 등록됩니다. 업체 직원은 별도로 일반(개인) 사용자로 가입한 뒤 사업자에 연결할 수 있습니다.
</Callout>

### 3단계: 본인인증

본인인증 방법을 선택합니다. **공인인증서** 또는 **PIN 인증** 중 하나를 선택하여 본인 확인을 완료합니다.

<AnnotatedImage src="unipass-id-verify-select" section="clearance" alt="본인인증 선택 화면" caption="본인인증 선택 후 다음" />

### 4단계: 업체 정보 입력

사업자등록증에 기재된 **업체상호**와 **사업자등록번호**를 입력한 후 **다음**을 클릭합니다.

<AnnotatedImage src="unipass-biz-info-input" section="clearance" alt="업체상호 및 사업자등록번호 입력" caption="업체상호, 사업자등록번호 입력 후 다음 선택" />

<Callout type="warning" title="사업자등록번호 유의사항">
사업자등록번호가 틀린 경우 사업자변경가 불리될 수 있습니다. 다시 가입하더라도 이미 등록된 사업자등록번호로는 재등록이 어렵습니다.
</Callout>

### 5단계: 업체 주소 및 연락처 입력

사업자등록증 상의 **업체 주소**와 **연락처**를 정확히 입력한 후 **다음**을 클릭합니다.

<AnnotatedImage src="unipass-address-input" section="clearance" alt="업체 주소 및 연락처 입력" caption="사업자등록증 상의 업체 주소 및 연락처 입력 후 다음 선택" />

### 6단계: 관할 세관 선택 및 공인인증서 등록

사업장 주소지의 **관할 세관**을 선택하고, 공인인증서를 등록합니다.

<AnnotatedImage src="unipass-customs-office-cert" section="clearance" alt="관할 세관 선택 및 공인인증서 등록" caption="사업장 주소지 관할 세관 선택 후 공인인증서 등록" />

<Callout type="tip">
세관은 수출 통관을 담당하는 **통관지 세관**이 아니라, **사업장 주소지 관할 세관**을 선택해야 합니다.

공인인증서 등록 방법:
1. 인증서 위치 선택
2. 인증서 비밀번호 입력
3. 사업자에 제공하는 사업자 인증서 선택
4. 인증서 암호 확인 → 확인 버튼 선택
</Callout>

### 7단계: 업태 유형 선택

세관에 영업 등록되어 있는 **업태 유형을 모두 선택**합니다.

<AnnotatedImage src="unipass-biz-type-select" section="clearance" alt="업태 유형 선택" caption="세관 영업 등록 업태 유형 전체 선택" />

<Callout type="warning">
해당 사항이 없는 경우에도 **수출품** 항목은 반드시 선택해야 합니다.

업태 유형 미선택 시 수출 신고가 정상 처리되지 않을 수 있습니다.
</Callout>

### 8단계: 등록 완료

발급 완료 메시지가 표시되면 **확인** 버튼을 클릭합니다. 이것으로 유니패스 사용자 등록이 완료됩니다.

<AnnotatedImage src="unipass-register-complete" section="clearance" alt="유니패스 등록 완료" caption="발급 완료 확인" />

## 통관고유부호 조회 및 신청

유니패스 사용자 등록이 완료되면 통관고유부호를 조회하거나 신청할 수 있습니다.

상단 메뉴에서 **업무지원 → 통관고유부호 → 통관고유부호 조회/신청**을 선택합니다.

<AnnotatedImage src="unipass-code-lookup-menu" section="clearance" alt="통관고유부호 조회/신청 메뉴" caption="업무지원 > 통관고유부호 > 통관고유부호 조회/신청" />

조회된 통관고유부호를 확인하여 스마트십 정식수출신고 서비스 신청 시 입력합니다.

---

수출 신고는 크게 **목록통관**과 **수출목록변환신고(전자상거래 수출신고)** 두 가지 방식으로 나뉩니다. 각 방식의 특징을 비교하여 적합한 방식을 선택하세요.

## 수출 신고 종류 비교

| 구분 | 목록통관 | 수출목록변환신고 (전자상거래 수출신고) |
| --- | --- | --- |
| **수출 신고 방법** | 간소화된 절차로 진행 | 정식 수출 신고 절차를 통해 진행 |
| **신고 명의** | 특송 업체명 (트랙스로지스) | 개인 또는 기업 |
| **관세 환급** | 적용되지 않음 | 적용 가능 |
| **재수입 면세** | 적용되지 않음 | 적용 가능 |
| **부가세 영세율** | 적용 가능 (해외 소포 수령증으로 대체 가능) | 적용 가능 |
| **주요 특징** | 비교적 간단한 절차, 소액 수출에 유리 | 수출 실적 인정 시 세금 혜택 및 환급 가능 |
| **신청 방법** | 별도 신청 필요 없음 | 스마트십을 통하여 사전 신청 필수 (승인 완료 후 입고되는 화물부터 적용) |
| **비용** | 별도 비용 없음 | 화물 건별 비용 발생 |

## 수출목록변환신고 신청 방법

메뉴 경로: **통관 → 정식수출신고 → 정식수출신고 대행 신청**

사업자 통관 고유부호를 기재한 후 서비스 신청하기를 진행합니다. 승인이 완료된 이후 입고되는 화물부터 정식 수출신고가 적용됩니다.

<AnnotatedImage src="export-declaration-apply-page" section="clearance" alt="정식 수출 신고 서비스 신청 화면" caption="통관 > 정식수출신고 > 정식수출신고 대행 신청 화면" />

<Callout type="info" title="사업자 통관부호란?">
수출 신고 시 필수로 기재해야 하는 부호로, 관세청 유니패스에 등록된 사업자의 통관부호입니다.

통관고유부호가 없는 경우 [통관고유부호 발급 방법](/ko/clearance/customs-unique-code) 페이지를 참고하여 먼저 발급받으세요.
</Callout>

<Callout type="warning">
사업자 정보가 변경된 경우 서비스를 철회한 후 재신청해야 합니다.
</Callout>

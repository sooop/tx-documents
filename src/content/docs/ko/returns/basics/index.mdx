<Callout type="info">

## 반품 픽업 신청

구매자의 반품 화물을 트랙스로지스 센터에서 직접 픽업하여 수거하는 방식입니다.

**픽업 가능 국가:** 한국, 일본, 싱가포르

**메뉴:** 주문 → 반품 주문 → 반품주문 등록

### 신청 방법

<AnnotatedImage src="return-pickup-request" section="returns" alt="반품 픽업 신청 화면" caption="반품 주문 등록 — 반품 픽업 신청" />

① **[반품 픽업을 요청합니다.] 선택**

② **반품 주문 및 수거지 주소 입력**
   - 기존 주문 검색으로 자동 입력 가능 (필요 시 수정)
   - 한국, 일본의 경우 각 국가 트랙스로지스 센터로 화물이 회수됩니다.
   - 일본의 경우 수거일을 사전에 확인했다면 [반품 수거일]에 입력합니다.

③ **[반품 주문 생성하기] 선택**

<Callout type="info" title="한국 픽업">
영업일 기준 1~3일 내로 택배 기사님이 수거지 정보로 물건을 회수할 예정입니다. 미집화 시 반품 픽업 신청을 다시 진행해 주셔야 합니다.
</Callout>

<Callout type="info" title="일본 픽업">
**수거일 확인 가능 시:** ② 단계에서 [반품 수거일]을 입력한 후 등록합니다.

**수거일 미확인 시:** 수거일을 입력하지 않고 등록하면 수거지 전화번호로 문자가 발송됩니다. 문자를 통해 수거일을 선택한 후 픽업이 시작됩니다.
</Callout>

## 직접 발송 반품 신청

구매자가 이미 반품 상품을 발송한 경우, 송장번호를 등록하여 처리합니다.

### 직접 발송 반품 가능 국가

현재 **일본**만 지원합니다.

반품 화물을 아래 주소로 발송해주세요.

> **(273-0017) 千葉県船橋市西浦3-4-2、北エントランス2F 国内CS部署**
> 연락처: 050-8882-6040

**메뉴:** 주문 → 반품 주문 → 반품주문 등록

### 신청 방법

<AnnotatedImage src="return-direct-send" section="returns" alt="직접 발송 반품 신청 화면" caption="반품 주문 등록 — 이미 송장번호가 있는 경우" />

① **[이미 송장번호가 있습니다.] 선택**

② **반품 주문 및 수거지 주소 입력**
   - 기존 주문 검색으로 자동 입력 가능 (필요 시 수정)

③ **[반품 주문 생성하기] 선택**

<Callout type="danger" title="일본 착불 택배 주의">
일본에서 **착불 택배**로 반품을 발송한 경우, 반드시 **센터 입고 전**에 반품 주문을 등록해야 합니다. 사전 등록 없이 도착한 착불 택배는 수령이 거부됩니다.
</Callout>

## 반품 주문 관리

반품 화물이 센터에 도착하면 반품 주문 관리 메뉴에서 상태를 확인하고, 재배송 또는 폐기를 요청할 수 있습니다.

**메뉴:** 주문 → 반품 주문 → 반품주문 관리

### 반품 주문 상태

| 상태 | 설명 |
|------|------|
| 센터로 배송 중 | 픽업 또는 구매자 발송 화물이 센터로 배송 중 (취소 가능) |
| 반품 입고 완료 | 센터 도착 후 처리 대기 중 (재배송 / 폐기 / 검수 요청 가능) |
| 검수 요청 | 상세 검수 진행 중 |
| 검수 완료 | 상세 검수 완료 |
| 다른 주소로 배송 | 재배송 진행 중 |
| 폐기 | 폐기 처리 진행 중 |
| 반품 처리 완료 | 모든 처리 완료 |

### 처리 방법

<AnnotatedImage src="return-management-list" section="returns" alt="반품 주문 관리 화면" caption="반품 주문 관리 — 목록 조회 및 처리 요청" />

① 검색 조건을 입력하여 반품 주문 목록을 조회합니다.

② 주문을 선택하면 도착 화물의 외관 사진 및 송장을 확인할 수 있습니다.

③ 원하는 처리 방식을 선택합니다.
   - **폐기 요청**: 반품 상품을 폐기 처리합니다.
   - **다른 주소로 배송하기**: 반품 상품을 다른 주소로 재배송합니다.

<Callout type="info" title="재배송 안내">
- **한국 주소로 재배송**: 일본 → 한국 해외 배송비가 TxMoney에서 자동 차감됩니다.
- **일본 주소로 재배송**: 일본 → 일본 배송비가 차감되거나, 수취인 부담 옵션을 선택할 수 있습니다.

재배송 신청 시 **출발국 창고 권한**이 필요합니다. 권한이 없는 경우 별도로 창고 이용을 신청해야 합니다.
</Callout>

### 수취인 부담 재배송

수취인이 직접 배송비를 결제하도록 요청할 수 있습니다.

<AnnotatedImage src="return-management-cod" section="returns" alt="수취인 부담 재배송 프로세스" caption="수취인 부담 재배송 처리 흐름" />

<AnnotatedImage src="return-management-detail" section="returns" alt="수취인 부담 재배송 신청 화면" caption="수취인 부담 재배송 신청" />

① **[다른 주소로 배송하기]** 선택

② 수취인 부담 재배송 안내를 확인합니다.

③ **[확인]** 선택

④ **[반품 요청 완료하기]** 선택

수취인에게 배송비 결제 요청 문자가 발송됩니다.

<AnnotatedImage src="return-management-reship" section="returns" alt="수취인 배송비 결제 화면" caption="수취인에게 발송되는 배송비 결제 화면" />

- **결제 유효기간**: 신청일로부터 5일
- **결제 수단**: 카드 또는 TxMoney

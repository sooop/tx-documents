SmartShip에 주문을 등록하려면 먼저 발송할 상품을 등록해야 합니다. 상품은 **개별 등록**과 **대량 엑셀 업로드** 두 가지 방식을 지원합니다.

<Callout type="info" title="상품 등록 경로">
상단 메뉴에서 **상품관리**로 이동한 뒤 **+ 등록** 버튼을 클릭하면 개별 등록 화면이 열립니다. 대량 등록은 같은 페이지의 **엑셀업로드** 버튼을 이용합니다.
</Callout>

## 개별 상품 등록

상품 등록 화면은 **상품정보**, **해외배송 통관정보**, **판매채널 상품정보** 세 개의 탭으로 구성되어 있습니다. 탭 순서대로 입력한 뒤 **등록** 버튼을 클릭합니다.

### 1단계: 상품정보

**상품정보** 탭에서 아이템 기본정보를 입력합니다.

<AnnotatedImage src="product-reg-basic-info" section="seller-shipment" alt="상품정보 탭 — 아이템 기본정보 입력 화면" caption="상품정보 탭 — 아이템 기본정보" />

| 항목 | 필수 | 설명 |
|------|:----:|------|
| TX 상품코드 | — | 시스템 자동 부여, 수정 불가 |
| 판매자 정의코드 | | 판매자가 직접 설정하는 내부 관리 코드 |
| 상품바코드 | | 바코드 번호 입력 후 **확인** 클릭 시 기존 상품 정보 자동 조회 |
| 상품명 | ✓ | 3자 이상 입력 |
| 옵션명 | | 색상·사이즈 등 옵션 구분이 필요한 경우 입력 |
| 가격 / 통화 | | 판매 단가와 통화 단위 선택 (기본: KRW) |
| 중량(kg) | | 0.1 이상의 숫자값 입력 |
| 크기(cm × cm × cm) | | 가로 × 세로 × 높이 |
| 원산지 | | 드롭다운에서 선택 |
| 소재 | | 소재 종류 선택 및 직접 입력 |
| 배터리 포함 | | 배터리 내장 여부 선택 (기본: 포함하지 않음) |
| 민감상품여부 | | 해당 없음 / 해당 선택 |
| HS 코드 | | 관세 분류 코드 직접 입력 |

### 2단계: 해외배송 통관정보

**해외배송 통관정보** 탭에서 수출 통관에 필요한 정보를 입력합니다.

<AnnotatedImage src="product-reg-customs-info" section="seller-shipment" alt="해외배송 통관정보 탭 입력 화면" caption="해외배송 통관정보 탭 — 상품 통관 정보 관리" />

| 항목 | 설명 |
|------|------|
| 통관상품명 | 수출 신고 시 사용되는 영문 품목명. **영문 대문자**로 입력 |
| 상품 바코드 | 바코드 번호 |
| 판매사이트 URL | 상품이 등록된 판매 페이지 URL |
| HS / HSN | 대분류 → 중분류 → 소분류 순으로 선택하거나 코드 직접 입력 |
| 소재 | 소재 종류 및 성분 비율 |
| 배터리 포함여부 | 배터리 내장 여부 |
| 민감상품여부 | 민감 품목 해당 여부 |
| 모델명 / 브랜드 | 제품 모델명 및 브랜드 |
| 원산지 / 제조사 | 생산 국가 및 제조사명 |
| 통관 상품명(zh-cn) | 중국 향 발송 시 중국어 품목명 입력 |
| 제품 세번 | 세관 신고용 세번 코드 |
| GST 세율(%) | GST 적용 국가(호주 등) 발송 시 입력 (기본: 0) |
| VAT ID | VAT 등록 번호 |
| EORI | 유럽 향 발송 시 EORI 번호 |

<Callout type="warning" title="통관상품명 주의">
통관상품명은 반드시 **영문 대문자**로 입력해야 합니다. 한글·특수문자 입력 시 통관 오류가 발생할 수 있습니다.
</Callout>

### 3단계: 판매채널 상품정보

**판매채널 상품정보** 탭에서 연동된 쇼핑몰별 상품 정보를 확인하고 매핑할 수 있습니다.

모든 정보 입력 후 **등록** 버튼을 클릭하면 상품 등록이 완료됩니다.

## 대량 상품 등록 (엑셀 업로드)

다수의 상품을 한 번에 등록할 때는 엑셀 업로드 기능을 사용합니다.

<AnnotatedImage src="product-bulk-reg" section="seller-shipment" alt="대량 배송상품 등록 — 엑셀 업로드 화면" caption="대량 배송상품 등록 화면" />

### 엑셀 양식 다운로드

1. 대량 배송상품 등록 화면 우측 상단의 **XLSX 포맷 다운로드하기** 버튼을 클릭합니다.
2. 다운로드된 엑셀 파일을 열어 상품 정보를 입력합니다.

<Callout type="warning" title="신규 양식 안내 (2024년 7월 11일 적용)">
신규 양식부터 칼럼명에 코드와 괄호가 포함되어 있습니다. 해당 코드·괄호를 **삭제하지 마세요**. 삭제할 경우 업로드가 정상 처리되지 않습니다.
</Callout>

### 엑셀 컬럼 안내

<Collapse title="전체 컬럼 설명 보기">

| 컬럼명 | 코드 | 설명 |
|--------|------|------|
| 상품명 | M00 | 상품명 (필수) |
| 발송 수량 | M01 | 발송 단위 수량 |
| 단가 | M02 | 상품 단가 |
| 통화 | M03 | 통화 코드 (예: KRW, USD, JPY) |
| 세부 분류 | M04 | 상품 세부 분류 |
| 옵션 | M05 | 옵션 정보 (색상, 사이즈 등) |
| 상품 URL 정보 | M06 | 판매 페이지 URL |
| 상품 소재(성분) | M07 | 소재 및 성분 정보 |
| HS/JAL | M08 | HS 코드 또는 JAL 코드 |
| Overseas / Trade code | — | 해외 거래 코드 (HS Code, JAL 등) |
| 상품 브랜드 | M10 | 브랜드명 |
| Shop Name | M11 | 판매 사이트명 |
| 바코드 | M12 | 상품 바코드 번호 |
| 행우세 | M13 | 행우세 적용 여부 |
| 행우세 코드(뒷부분) | M14 | 행우세 코드 뒷자리 |
| 사용자 정의 상품 코드 | M15 | 판매자 내부 관리 코드 |
| GST Rate | M16 | GST 세율 (%) |

</Collapse>

### 파일 업로드

1. 드롭다운에서 **등록**을 선택합니다.
2. **파일 선택하기** 버튼을 클릭하여 작성한 엑셀 파일을 선택합니다.
3. 업로드 후 **처리 결과(ST0)** 와 **상세 내역 정보(ST1)** 컬럼에서 등록 결과를 확인합니다.

<Callout type="tip" title="업로드 실패 시">
처리 결과(ST0)가 오류로 표시되면 상세 내역 정보(ST1)를 확인해 원인을 파악하고 해당 항목을 수정한 뒤 재업로드하세요.
</Callout>

## 등록된 상품 확인 및 관리

상품 등록 후 **상품관리** 메뉴의 상품조회에서 목록을 확인합니다.

<AnnotatedImage src="product-management" section="seller-shipment" alt="상품관리 — 상품조회 화면" caption="상품관리 — 상품조회 목록" />

- **키워드 검색**: 상품코드, 상품명 등 조건으로 필터링
- **조회내용 다운로드**: 현재 조회된 목록을 엑셀 파일로 내보내기
- **엑셀업로드**: 등록된 상품 정보를 대량으로 수정할 때 사용

<Callout type="tip" title="상품 정보 일괄 수정">
등록된 상품의 정보를 수정하려면 **엑셀업로드** 클릭 후 드롭다운에서 **수정**을 선택하고, **수정용 포맷 다운로드** 버튼으로 현재 등록된 상품 데이터를 받아 편집한 뒤 재업로드하세요.
</Callout>

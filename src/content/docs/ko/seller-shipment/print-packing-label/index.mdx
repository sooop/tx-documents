SmartShip에 주문을 등록한 후 센터 입고 시 화물과 주문을 인식하기 위한 라벨입니다. 박스에 라벨을 부착하지 않거나 잘못 부착한 채 센터로 보내시는 경우 입고가 불가능합니다.

<Callout type="info" title="메뉴 경로">
- **송장 > 송장 생성 및 출력**: [바로가기](https://smartship2.tracxlogis.com/waybills)
- 주문 > 주문 관리 상단 탭에서도 라벨을 출력할 수 있습니다.
</Callout>

## 라벨 출력 방법

### 출력 실행

<AnnotatedImage src="waybills-select-orders-01" section="seller-shipment" alt="라벨 출력 주문 목록 화면" caption="송장 생성 및 출력 페이지 — 출발국가별로 분류된 주문 목록" />

1. 라벨 출력 가능한 주문이 출발국가별로 분류되어 있습니다. 출력할 구간을 선택합니다.
2. 액션 메뉴에서 **라벨 출력 → 전체 주문**을 선택합니다. 특정 주문만 출력하려면 해당 주문들을 체크한 후 **선택된 주문**을 선택합니다.

### 출력 설정

<AnnotatedImage src="waybills-print-barcode-label-01" section="seller-shipment" alt="라벨 출력 설정 대화상자" caption="라벨 출력 설정 창 — 프린터, 용지, 인쇄 항목, 정렬 기준을 설정합니다." />

1. **출력 방식과 용지 크기**를 선택합니다. A4 용지 6칸·8칸·2칸 중 하나를 고르거나, 라벨 프린터기를 선택할 수 있습니다.
2. **라벨 출력 데이터**를 선택합니다. 판매사이트 주문 정보 또는 스마트십 등록 상품 정보 중 하나를 선택합니다.
3. **인쇄 항목**을 선택합니다. 주문번호, 상품명, SKU No. 등 라벨에 인쇄할 항목을 체크합니다.
4. **정렬 기준**을 선택합니다. 최대 2가지 기준을 조합할 수 있습니다.

설정 완료 후 **[저장 후 출력]** 버튼을 클릭합니다.

### 판매사이트 발송 처리

API 연동 주문이 있는 경우, 라벨 인쇄 후 판매사이트 주문을 발송 처리합니다.

<Callout type="warning" title="해외 배송 시 주의">
해외 배송의 경우 사전에 발번된 배송사가 없다면 TracX Logis로 배송사가 지정되며, 실제 현지 배송사 송장으로는 등록되지 않습니다.

입고 전 미리 발송 처리하면 판매채널로부터 가송장 등록으로 간주되어 불이익을 받을 수 있습니다.

API 연동 주문은 **센터 입고 시점에 자동으로 송장 등록 처리**됩니다.
</Callout>

## 라벨 부착 시 주의사항

<AnnotatedImage src="waybills-label-bad-cases" section="seller-shipment" alt="바코드 라벨 잘못 부착 사례" caption="잘못된 라벨 부착 사례 — 이와 같이 부착하면 입고가 불가하거나 지연될 수 있습니다." />

- 입고 시 자동 스캐너로 입고가 진행됩니다. 라벨 인쇄 상태가 양호하지 않거나 부착 위치가 올바르지 않으면 입고가 불가하거나 지연될 수 있습니다.
- 바코드와 QR코드는 번짐·흐림·잘림 없이 선명하게 인쇄해야 합니다.
- 인쇄 비율은 **100%**로 설정하세요. 축소 인쇄 시 인식이 불가할 수 있습니다.
- 라벨은 **박스의 가장 넓은 면**에 부착해야 합니다.
- 라벨이 모서리에 닿거나 접히거나 구겨지지 않도록 부착하세요.

주문 등록은 **개별 등록**, **엑셀 등록**, **API 연동** 세 가지 방식을 지원합니다. 편하신 방법으로 진행하세요.

<Callout type="info" title="메뉴 경로">
- [주문관리 바로가기](https://smartship2.tracxlogis.com/orders/new)
</Callout>

## 합포장 기준

**엑셀 등록** 및 **API 연동**으로 등록된 주문 중, 아래 세 가지 조건이 모두 일치하는 주문건은 자동으로 합포장됩니다.

**참조 주문번호 = 수취인명 = 연락처(HP)**

<Callout type="tip" title="참조 주문번호란?">
각 판매 사이트에서 제공하는 합포장 기준값을 입력합니다. 장바구니 번호, 카트번호, 배송번호 등이 이에 해당합니다.
</Callout>

---

## 1. 개별 등록

주문건을 한 건씩 직접 입력하여 등록하는 방식입니다.

1. 주문 관리 페이지 우측 상단에서 **[+ 주문 등록] → [개별 주문등록]** 을 선택합니다.

   <AnnotatedImage src="register-order-menu" section="seller-shipment" alt="주문 등록 방식 선택 메뉴" caption="[+ 주문 등록] 클릭 시 나타나는 등록 방식 선택 메뉴" />

2. **서비스 타입**에서 **판매자 출고**를 선택합니다.

   <AnnotatedImage src="orders-new-create-order-panel-01" section="seller-shipment" alt="개별 주문 등록 입력 화면" caption="개별 주문 등록 패널 — 서비스 타입에서 판매자 출고를 선택합니다" />

3. **수취인 정보**를 입력합니다. 올바른 우편번호를 입력하면 즉시 검색 결과가 표시되며, 마우스로 선택하면 주소가 자동 입력됩니다. 싱가포르의 경우 우편번호 입력 시 주소가 자동으로 입력됩니다.

   <AnnotatedImage src="order-new-search-postal-code" section="seller-shipment" alt="우편번호 검색 및 주소 자동완성 화면" caption="우편번호 입력 시 즉시 검색 결과가 표시됩니다. 목록에서 선택하면 주소가 자동 입력됩니다." />

4. **상품 정보**를 입력합니다. 상품명 입력 필드 앞의 검색 버튼을 통해 등록된 상품을 검색할 수 있으며, 기존 등록 상품의 상품명을 세 글자 이상 입력하면 자동으로 검색하여 빠르게 입력할 수 있습니다.

   <AnnotatedImage src="orders-new-create-search-item" section="seller-shipment" alt="등록상품 검색 화면" caption="검색 버튼 클릭 또는 세 글자 이상 입력 시 등록된 상품을 빠르게 검색할 수 있습니다." />

5. 하단 **[등록하기]** 버튼을 클릭하면 등록이 완료됩니다. 완료 알림 화면에서 QSP no.를 복사할 수 있습니다.

## 2. 엑셀 등록

여러 주문건을 엑셀 파일로 한 번에 업로드하는 방식입니다.

<Callout type="info" title="해외 발송 시 상품 URL 입력 안내">
해외 배송 주문의 경우 통관을 위한 **상품 URL** 정보가 필수입니다. 단, 사전에 등록된 상품의 경우 **사용자 정의 상품코드(O25)** 만 입력하면 이전에 등록된 상품의 URL 정보를 사용할 수 있으며, 이 경우 URL은 기입하지 않아도 됩니다.
</Callout>

### 2-1. 표준 엑셀 양식

SmartShip에서 제공하는 기본 양식을 사용합니다.

1. 주문 관리 페이지 우측 상단에서 **[+ 주문 등록] → [엑셀 주문등록]** 을 선택합니다.

   <AnnotatedImage src="orders-new-upload-excel" section="seller-shipment" alt="엑셀 주문 등록 화면" caption="[+ 주문 등록] → [엑셀 주문등록] 선택 후 양식을 선택합니다" />

2. **[표준 엑셀 양식]** 을 선택합니다.
3. **[템플릿 다운로드]** 를 선택하여 양식 파일을 다운로드합니다.
4. 다운로드된 엑셀 파일에 필요한 항목을 채웁니다.
5. 업로드 양식에서 **[스마트십 엑셀 파일]** 을 선택한 후, **[파일 선택하기]** 버튼을 클릭하여 파일을 업로드합니다.
6. 업로드 후 오류가 없는지 확인합니다.
7. 하단 **[주문 등록]** 버튼을 클릭하면 등록이 완료됩니다.

### 2-2. 주요 쇼핑몰 엑셀 양식

쇼핑몰에서 내보낸 엑셀 파일을 그대로 업로드할 수 있습니다.

1. 주문 관리 페이지 우측 상단에서 **[+ 주문 등록] → [엑셀 주문등록]** 을 선택합니다.
2. **[주요 쇼핑몰 엑셀양식]** 을 선택합니다.
3. 사이트를 선택합니다.
4. 해당 사이트에서 다운로드 받은 엑셀 파일을 업로드합니다.
5. **[주문 등록]** 버튼을 클릭하여 주문을 등록합니다.

### 2-3. 나만의 엑셀 양식

<Callout type="tip" title="표준 양식 사용을 권장합니다">
**나만의 엑셀 양식** 대신, 스마트십 **표준 엑셀 양식**을 사용하세요. 표준 양식의 컬럼 코드(예: O01)만 유지하면 컬럼 이름이나 순서를 자유롭게 변경하거나 불필요한 컬럼을 삭제할 수 있습니다. 원하는 형식으로 템플릿 파일을 편집한 뒤, **표준 양식**으로 업로드하는 방식을 권장합니다.
</Callout>

## 3. API 연동 등록

API 연동을 통해 쇼핑몰 주문을 자동으로 수집하는 방식입니다.

- API 연동 설정 시 **자동 주문 수집**을 사용하면 주문이 자동으로 등록됩니다.
- 수동으로 API 주문 연동을 실행하려는 경우에도 사용할 수 있습니다.

API 연동 설정 방법은 [API 연동 가이드](/ko/api-guide/)를 참조하세요.

## 주문 등록 확인

판매자 출고 타입으로 등록된 주문은 주문 관리 화면 상단의 **상태별 집계** 중 **판매자 출고** 영역에서 확인할 수 있습니다.

<AnnotatedImage src="orders-new-see-detail" section="seller-shipment" alt="주문 상태별 집계 및 주문 목록 화면" caption="상태별 집계의 판매자 출고 영역에서 등록된 주문을 확인할 수 있습니다." />

각 행의 **[상세 보기]** 액션 버튼을 클릭하여 주문의 세부 내용을 확인할 수 있습니다.

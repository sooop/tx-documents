배송할 상품 정보와 박스 크기를 정확하게 입력해야 합니다. 잘못 입력하면 통관 지연 또는 추가 요금이 발생할 수 있습니다.

## 배송 상품 정보 입력

<AnnotatedImage src="smart-post-product-info" section="smart-post" alt="상품 정보 입력 화면" caption="Step 2 — 배송 상품 정보 입력 화면" />

| 항목 | 설명 |
|------|------|
| **상품가격 / 통화** | 모든 상품은 동일한 통화로 입력. 자동 환산되지 않으므로 직접 확인 필요 |
| **상품명** (필수) | 영문·숫자·특수문자 입력 가능. 예: T-shirt, Bag, Accessories, Snacks |
| **수량 및 단가** (필수) | 총금액은 수량 × 단가로 자동 계산됨 |
| **HS/HSN 코드** | 국제 통관용 품목 코드 (10자리) |
| **상품 상세 URL** (필수) | 해당 상품의 상세 페이지 링크 |
| **상품 대표 이미지** (필수) | 배송 및 통관 확인 시 필수 자료 |

<Callout type="warning" title="주의사항">
상품명과 URL은 반드시 동일한 상품 정보여야 합니다. 잘못된 정보 기재 시 세관 문제가 발생할 수 있습니다.
</Callout>

## 박스 무게 & 부피 입력

실제 박스의 무게와 크기를 입력합니다. **실제 무게와 부피무게 중 큰 값**을 기준으로 운임이 계산됩니다.

<AnnotatedImage src="smart-post-box-size" section="smart-post" alt="박스 무게 및 부피 입력 화면" caption="Step 2 — 박스 무게 및 부피 입력 화면" />

### 택배 중량 (kg)

저울로 측정한 박스의 **실제 무게**를 입력합니다. 소수점 입력이 가능합니다. (예: 1.25)

### 부피무게

포장 완료된 박스의 가로·세로·높이를 cm 단위로 입력하면 부피무게가 자동 계산됩니다.

```
부피무게(kg) = (가로 × 세로 × 높이) ÷ 6,000
```

**계산 예시:**
- 박스 크기: 40cm × 30cm × 20cm
- 부피무게: 40 × 30 × 20 ÷ 6,000 = **4kg**
- 실제 무게 3kg < 부피무게 4kg → 운임은 **4kg 기준** 적용

<Callout type="warning" title="주의사항">
- 실제 측정값을 입력해야 합니다. 부정확하면 추가 요금이 발생할 수 있습니다.
- 부피는 **포장 완료 상태** 기준으로 측정하세요.
</Callout>

<Callout type="tip">
국제운송에서는 부피무게가 더 큰 경우가 많으므로, 정확한 치수 측정이 중요합니다.
</Callout>

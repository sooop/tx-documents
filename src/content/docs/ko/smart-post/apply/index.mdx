택배 신청은 PC 웹 또는 SmartPost 모바일 앱에서 진행합니다.

## 접속 방법

| 접속 환경 | 방법 |
|-----------|------|
| **PC** | tracxlogis.com 접속 → 서비스 → 스마트포스트 |
| **모바일** | SmartPost 애플리케이션 → 택배 신청 |

---

## 1단계: 발송인 정보 입력

픽업 방문 및 통관 서류 작성에 사용되므로 **정확한 정보**를 입력해야 합니다.

<AnnotatedImage src="smart-post-apply-sender" section="smart-post" alt="발송인 정보 입력 화면" caption="Step 1 — 발송인 정보 입력 화면" />

| 항목 | 설명 |
|------|------|
| **발송인 이름** | 실제 발송자의 실명을 영문으로 입력 |
| **발송인 주소** | 픽업 기사 방문을 위해 상세주소까지 정확히 기재 |
| **발송인 전화번호** | 국제번호 자동 설정(+82) / 픽업 및 추가배송비 안내용 SMS 수신 가능 번호 |
| **발송인 이메일** | 주문 안내 및 배송 알림 수신용 |
| **방문 픽업 요청 날짜** | 택배사 사정에 따라 실제 픽업 일자는 조정될 수 있음 |

<Callout type="warning" title="주의사항">
- 영어 주소는 반드시 한글 주소와 동일한 위치를 기재해야 합니다.
- 입력한 연락처로 픽업 및 배송 안내가 발송됩니다.
- 상품정보와 발송인 정보가 일치하지 않으면 픽업 및 통관이 지연될 수 있습니다.
</Callout>

---

## 2단계: 수취인 정보 및 서비스 선택

<AnnotatedImage src="smart-post-apply-recipient" section="smart-post" alt="수취인 정보 입력 화면" caption="Step 1 — 수취인 정보 및 서비스 선택 화면" />

| 항목 | 설명 |
|------|------|
| **수취인 이름** | 해외 배송 시 여권 또는 현지 신분증의 영문 이름으로 기재 |
| **수취인 주소** | 해외 주소는 반드시 **영문 표기** |
| **수취인 전화번호** | 국제번호 선택 / 현지 배송사 확인용으로 유효한 번호 입력 |
| **수취인 이메일** | 배송 진행 알림 수신용 |

<Callout type="warning" title="주의사항">
- 주소 및 연락처 불일치 시 배송 실패 가능성이 높아집니다.
- 모든 정보는 **실제 수취인 정보**로 입력해야 합니다.
</Callout>

<Callout type="tip">
모든 단계에서 정확한 정보를 입력해야 배송 지연과 추가 비용을 방지할 수 있습니다.
</Callout>

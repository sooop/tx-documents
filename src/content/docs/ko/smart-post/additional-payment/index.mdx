---

픽업 후 집하 센터에서 실측한 무게·부피가 최초 입력값보다 큰 경우, 국제 특송 규정에 따라 **실측값 기준으로 운임이 재산정**되어 추가 배송비 결제가 필요합니다.

<Callout type="warning" title="미결제 시 반송 처리">
추가 요금 안내를 받은 후 **영업일 5일 이내** 결제하지 않으면 **착불 반송**됩니다. 픽업료를 제외한 해외 운송료는 Tx-money로 환불됩니다.
</Callout>

## 추가 결제 프로세스

1. 추가 결제 안내 **SMS / 카카오톡 알림 / 앱 푸시**의 링크를 클릭합니다.
2. 결제 페이지에서 추가 요금을 확인합니다.
3. 결제 방식을 선택하고 결제합니다.
4. 결제 완료 후 배송이 정상 진행됩니다.

## 결제 방법

| 결제 수단 | 설명 |
|-----------|------|
| **카드 결제** | Stripe 보안 게이트웨이를 통해 안전하게 결제. 브라우저 팝업 허용 필요 |
| **TxMoney** | 사전 충전된 잔액이 있는 경우 사용 가능 |

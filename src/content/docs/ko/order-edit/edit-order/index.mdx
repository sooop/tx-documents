주문 등록 후 수정하기 기능을 통해 수정할 수 있습니다. 주문 진행 상태에 따라 일부 항목을 수정할 수 없는 경우도 있습니다.

<Callout type="info" title="메뉴 경로">
- [주문관리 바로가기](https://smartship2.tracxlogis.com/orders/new)
</Callout>

<Callout type="warning" title="TX 3PL 풀필먼트 주문 수정 제한">
TX 3PL 풀필먼트 주문의 경우, 센터로 출하지시를 내린 이후에는 상품 변경/추가/삭제가 허용되지 않습니다. SmartShip은 이러한 주문에 대해 자동으로 출하지시를 취소한 후 수정할 수 있도록 동작합니다.

이미 피킹이 진행되어 출하지시 취소를 할 수 없는 경우에는 수정이 불가합니다. 이 경우 기존 주문을 삭제하신 후 재연동/재등록해야 합니다.
</Callout>

## 주문 수정 방법

1. 수정이 필요한 주문을 선택합니다.
2. 각 행의 액션 메뉴에서 **[수정하기]** 를 선택합니다.
3. 주문의 기본정보, 수취인정보, 상품 정보 등을 수정한 후 저장합니다.
4. 풀필먼트 주문의 경우 저장 시, 출하지시를 동시에 처리할 수 있습니다.

---

## 다건 연속 수정 방법

여러 주문을 하나씩 순서대로 수정할 수 있는 방법입니다.

1. 수정이 필요한 주문들을 체크하여 모두 선택합니다.
2. 데이터 그리드 상단 메뉴에서 **[일괄편집 → 주문 수정]** 을 선택합니다.

   <AnnotatedImage src="orders-new-bulk-edit-select" section="seller-shipment" alt="일괄편집 메뉴에서 주문 수정 선택" caption="데이터 그리드 상단 [일괄편집] 메뉴에서 [주문 수정]을 선택합니다." />

3. 주문 수정 패널이 표시되며, 왼쪽 사이드 메뉴에 선택한 주문 목록이 표시됩니다.
4. 주문 내용을 변경 중 다른 주문을 선택하면, 변경 사항이 남은 주문은 **노란색**으로 표시됩니다.

   <AnnotatedImage src="not-saved-order" section="seller-shipment" alt="미저장 주문이 노란색으로 표시된 수정 패널" caption="변경 사항이 있으나 아직 저장하지 않은 주문은 노란색으로 표시됩니다." />

5. 변경 사항을 저장한 주문은 **녹색**으로 표시됩니다. 이를 통해 수정을 완료한 주문과 수정하지 않은 주문을 쉽게 구분할 수 있습니다.

   <AnnotatedImage src="saved-order" section="seller-shipment" alt="저장 완료 주문이 녹색으로 표시된 수정 패널" caption="저장을 완료한 주문은 녹색으로 표시됩니다." />

## 주문 속성 일괄 변경

여러 주문의 특정 속성을 동시에 변경하는 방법입니다.

<Callout type="tip">
데이터 그리드에서 첫 주문을 선택한 후 마지막 주문을 **Shift** 키를 누른 채 클릭하면 연속 선택할 수 있습니다.
</Callout>

1. 여러 주문을 선택합니다.
2. 상단 메뉴의 **[일괄편집 → 상품속성 일괄변경]** 을 선택합니다. (위 이미지의 ② 항목)

3. 배송 등급 등 일괄로 변경하려는 속성을 수정합니다.
4. **[저장하기]** 버튼을 클릭합니다.

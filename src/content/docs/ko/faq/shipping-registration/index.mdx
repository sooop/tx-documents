## 배송 등록이 안돼요 (엑셀 업로드 오류)

### 상품 정보 불일치

상품 등록 시 입력한 상품명과 배송 등록 시 입력하는 상품명이 **띄어쓰기까지 정확히 일치**해야 합니다.

상품 정보 관리에 옵션이 등록되어 있다면, 배송 등록 엑셀의 Item Option 항목에 동일한 옵션을 입력해야 합니다.

### LBL000123 오류 — "입력 값을 확인하십시오"

데이터에 **특수문자**가 포함되어 있을 때 발생합니다. 해당 특수문자를 삭제하면 해결됩니다.

### LBL000092 오류

배송 불가 국가를 입력한 경우 발생합니다. 발송 가능 국가인지 확인 후 올바른 국가를 입력하세요.

## 엑셀 업로드 양식 사용 방법

스마트십 엑셀 양식의 컬럼명과 순서는 자유롭게 편집할 수 있습니다. 단, **컬럼 코드는 반드시 유지**해야 합니다.

| 항목 | 가능 여부 |
|------|----------|
| 칼럼명 변경 | 가능 (코드 제외) |
| 순서 변경 | 가능 |
| 컬럼 삭제 | 가능 |
| 코드 삭제 | **불가** — 해당 컬럼 인식 안 됨 |
| 코드 변경 | **불가** — 데이터 오입력 발생 |

<Callout type="info" title="이전 버전 양식 안내">
2024년 7월 11일 이전 양식(컬럼 코드 미포함)은 신규 제공되지 않지만, 보유 중인 경우 계속 사용 가능합니다. 단, 해당 양식은 컬럼 순서 변경이 지원되지 않습니다.
</Callout>

## 배송 취소 가능한가요? (이미 발송한 주문)

### 이미 입고된 화물

이미 입고되어 송장번호가 생성된 화물은 **발송 보류 및 회수가 불가**합니다.

### 미입고 화물

아직 입고되지 않은 화물이라면 **배송 중단** 처리 후 일정 기간 보관 후 반송됩니다(약 7~10일 소요).

**배송 중단 방법**: [배송] > [배송 중] > 주문 선택 > [배송 중단 요청하기]

## 일본향 배송 등록이 실패되었어요

일본 세관 규정에 따라 다음 조건에서는 배송 등록이 불가능합니다.

| 조건 | 기준 | 해결 방법 |
|------|------|----------|
| 상품 개수 초과 | 패킹 내 상품 총합 25개 이상 | 한 박스에 24개 이하로 분할 포장 |
| 최소 상품 금액 미달 | 상품 개별 금액이 JPY 100 미만 | 샘플·사은품도 JPY 100 이상으로 입력 |

## 일본향 통관 불가 알림이 떴어요

JP향 주문이 등록되었을 때 **통관 불가 및 관세 발생 예상 건을 사전에 알려주는 기능**입니다. 알림이 표시되어도 배송 자체에는 지장이 없습니다.

주문 관리 화면 상단에 안내 메시지와 [대상 주문 보기] 버튼이 표시됩니다.

### 알림 발생 기준

1. 한 패킹의 상품가가 **JPY 16,666 초과** 시
2. 동일 수취인 주문들의 상품가 합계가 **JPY 16,666 초과** 시
3. 동일 수취인 주문의 수량 합계가 **25개 이상** 시

<Callout type="info">
관세 발생 후보를 사전 안내하는 기능이므로, 실제 관세 발생과 다를 수 있습니다. 일본 세관 판단에 따라 예상치 못한 관세가 발생할 수도 있습니다.
</Callout>

## 주문 오류/처리 불가인 경우 어떻게 해야 하나요?

상품 수량, 금액, SKU를 수정해야 하는 경우 주문이 **주문 오류/처리 불가** 상태로 분류됩니다.

해당 주문을 선택한 후 **[주문 수정하기]** 버튼을 클릭하여 수정하세요.

## 포장 시 주의사항을 알려주세요

<Callout type="danger" title="포장 불충분 시 보상 불가">
포장 불충분으로 인한 파손 또는 분실 시 손해배상이 불가합니다. 폴리백 포장의 경우에도 파손에 대한 보상 청구가 불가합니다.
</Callout>

### 올바른 포장 방법

1. **Fragile 스티커 부착** — 파손 주의 상품에 필수
2. **완충재로 빈 공간 채우기** — 에어캡 등을 사용해 박스 내부의 빈 공간을 모두 채우기
3. **모서리 정리** — 모서리를 타이트하게 정리하여 돌출 부분 없애기

<Callout type="info">
자동 스캐너(레이저 측정)를 사용하므로 동일한 사이즈 박스도 측정값이 다를 수 있습니다.
</Callout>

## 바코드라벨 부착을 어떻게 해야 하나요?

- 하나의 박스(폴리백)에 **동일한 패킹번호의 라벨만** 부착해야 합니다.
- 바코드 번호는 시스템에서 한 번만 인식됩니다. 두 개 이상의 박스에 같은 바코드번호를 부착하면 한 박스만 입고 처리됩니다.
- 국내 택배 송장과 바코드라벨을 함께 부착할 때는 바코드라벨이 가려지지 않도록 하세요.
- 여러 수취인의 주문을 한 번에 보낼 때는 각 패킹에 바코드라벨을 부착한 후 큰 박스에 넣고 택배 송장을 부착하세요.

<AnnotatedImage src="barcode-label-invalid" section="faq" alt="바코드라벨 인식 불가 예시" caption="인식 불가 바코드라벨 예시" />

### 바코드 인쇄 및 부착 기준

- 바코드 및 QR코드는 번짐, 흐림, 잘림 없이 선명하게 인쇄해야 합니다.
- 출력 비율은 반드시 **100%** 로 설정하세요 (축소 인쇄 불가).
- 바코드라벨은 **가장 넓은 면**에 부착해야 합니다.

## 한국발 해외향 보상 기준이 궁금해요

보상 기준은 배송 유형(이코노미/스탠다드)과 포장 주체(판매자/풀필먼트)에 따라 다릅니다.

- **이코노미**: 우편함 투함 배송
- **스탠다드**: 일반 배송
- **판매자**: 판매자가 직접 포장 후 배송 의뢰
- **풀필먼트**: 트랙스로지스 창고에서 3PL 출고

| 사고 유형 | 보상 여부 |
|----------|----------|
| 폴리백 포장 파손 | 조건부 보상 |
| 폴리백 분실 | 보상 가능 |
| 내부 상품 파손 (완충재 부족) | 보상 불가 |
| 배송완료 미수취 | 조건부 보상 |
| 폴리백 내품 탈락 | 조건부 보상 |
| 누액 파손 | 보상 불가 |
| 현지 송장 오부착 (교차 출고) | 보상 가능 |
| 원터치(조립식) 박스 파손 | 보상 불가 |

세부 기준은 [채팅상담](https://livechat.tracxlogis.com) 또는 help-kr@tracxlogis.com으로 문의해 주세요.

## 유류할증료가 어떻게 되나요?

국제 유가에 따른 유류할증료가 배송비에 추가됩니다.

- **반영 주기**: 당월 16일 ~ 익월 15일 적용
- **적용 기준**: 한국 출발 또는 입고 일자 기준
- **계산 방법**: 1kg당 유류할증료 기준

### 적용 구간

| 구간 | 국가 |
|------|------|
| 중거리 | 일본, 싱가포르, 대만, 홍콩, 중국 |
| 장거리 | 미국 |

<Callout type="info">
미국, 대만, 중국은 배송 요율에 유류할증료가 포함(ALL-IN-1)되어 있습니다. K-packet, EMS는 우체국 특별 운송 수수료에 따라 변동될 수 있습니다.
</Callout>

현재 유류할증료 확인: [배송비 계산기](https://smartship.tracxlogis.com/DeliveryOrder/RateCalculatorV2)

## 연동된 API를 해제하려면 어떻게 하나요?

### API 해제 방법

**[주문] > [쇼핑몰 연동] > [쇼핑몰 API 설정]**

1. 해제할 API 선택
2. **[연동 해제]** 버튼 클릭
3. **[OK]** 클릭 → 해제 완료

### 연동된 주문 삭제 방법

**[주문] > [주문 관리]**

1. 주문이 등록된 탭 선택
2. 삭제할 주문 선택 후 **[삭제하기]** 클릭

<Callout type="warning">
배송 중인 주문은 삭제할 수 없습니다. 배송 완료 후에만 API를 해제할 수 있습니다.
</Callout>

- **어드민**: 부계정 중 가장 광범위한 메뉴 접근 가능

## 큐탠재팬 도착보장 서비스 (韓ダッシュ, 칸닷슈) 이용 가이드

Qoo10 JP(이베이 재팬)와 TracX Logis가 운영하는 배송 도착보장 서비스입니다. eBay 공식 물류 파트너를 통해 제공되며, 예상 도착일을 사전에 안내하고 해당 일자까지의 정확하고 신속한 배송을 보장합니다.

### 주요 장점

**안정적인 재고관리**
- 도착보장 전용 재고 별도 관리
- 실시간 연동으로 과다 판매 방지
- 도착보장 주문에 자동 할당 및 우선 처리

**특별 코너 전시**
- eBay 전용 기획전 및 검색 상단 우선 노출
- 검색 결과에 도착보장 태그 및 날짜 강조 표시

**빠른 출고**
- 14:00 이전 결제 완료 주문은 익일 출고 보장

### 서비스 신청 방법

**1단계 — 스마트십 계정 확인**

- **기존 이용자**: [사용 중인 스마트십 계정이 있습니다] 선택 → JQSM 아이디와 회원번호 입력 → Qoo10 Japan 계정으로 로그인
- **신규 가입**: [스마트십 신규 가입] 선택 → JQSM 아이디와 회원번호 입력 → 가입 진행

**2단계 — 칸닷슈 서비스 신청**

1. [칸닷슈 서비스를 위한 정보 제공 동의] 확인 후 동의
2. 서비스를 사용할 샵 ID 선택 (Qoo10 ID가 2개 이상 연동된 경우)
3. 판매자 인증키 입력 (QSM에서 칸닷슈 서비스 신청 시 발급)
4. [칸닷슈 서비스 사전 신청하기] 선택
5. 풀필먼트 미이용 고객은 이용 계약서 동의 팝업 확인 → TracX Logis 승인 대기

**3단계 — SKU 생성 및 재고 입고**

- SKU 생성: [풀필먼트 > SKU 생성하기](/ko/fulfillment/create-sku/) 가이드 참고
- 입고 신청: [풀필먼트 > 입고 신청하기](/ko/fulfillment/request-inbound/) 가이드 참고

서비스 상세 안내: [TracX Logis 칸닷슈 페이지](https://smartship2.tracxlogis.com/efa)

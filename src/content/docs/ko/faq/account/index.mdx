## ID를 변경할 수 있나요?

스마트십 로그인 ID는 변경할 수 있습니다.

<Callout type="warning" title="변경 전 주의사항">
- 이메일이 아닌 형식의 ID는 이메일 주소로만 변경 가능합니다.
- ID 변경 후 90일이 지나야 다시 변경할 수 있습니다.
- 변경된 이메일로 트랙스로지스의 모든 공지 및 안내가 발송됩니다.
</Callout>

### 변경 방법

**[설정] > [내 정보] > [내 계정](https://smartship.tracxlogis.com/Customer/Account)**

1. 로그인 이메일 옆 **[수정]** 선택
2. 새 이메일 주소 입력
3. **[인증 메일 발송]** 클릭
4. 수신된 인증 메일("[TracxLogis] 로그인 이메일을 변경해주세요") 내 링크 클릭
5. 링크 클릭 후 자동 로그아웃되면 변경 완료

<Callout type="danger">
확인 팝업에서 [확인]을 눌러도 변경이 완료되지 않습니다. 반드시 이메일 내 인증 링크를 클릭해야 최종 변경됩니다.
</Callout>

---

## 회원가입이 안돼요

가입 시 모든 입력 항목의 빨간 점이 **파란 체크**로 바뀌어야 가입이 가능합니다. 입력된 내용을 다시 확인해 주세요.

- 상세 주소는 최소 한 글자 이상 반드시 입력해야 합니다.

---

## 비밀번호를 잊어버렸어요 / 비밀번호를 새로 설정하고 싶어요

로그인 페이지의 **비밀번호 찾기**를 통해 재설정할 수 있습니다.

<Callout type="info" title="이메일이 중복된 계정이 있는 경우">
초기 가입자의 경우 같은 이메일로 여러 계정 ID가 존재할 수 있습니다. 이때는 비밀번호 찾기 시 아이디도 함께 입력해야 합니다.

이메일 주소로 로그인하는 계정이라면, 아이디 입력란에 이메일 주소를 한 번 더 입력하세요.
</Callout>

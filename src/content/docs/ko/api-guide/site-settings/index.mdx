풀필먼트 서비스(TX 3PL) 이용 시, 판매사이트 주문의 상품에 대해 어떤 SKU가 출고되어야 하는지 구분하려면 각 채널 판매자 어드민에서 상품 및 옵션에 SKU와 연결할 수 있는 정보를 등록해두셔야 합니다.

## 판매채널 상품에 입력해야 하는 정보

다음 중 하나의 정보를 판매채널 상품 혹은 옵션에 기입해주세요. 각 판매채널별 상품/옵션 설정 방법은 개별 사이트 API 연동 가이드를 참고해주세요.

- **스마트십 SKU no.** — `700************-000**` 형식
- **공산품 바코드 번호** — 바코드 번호를 '판매자상품코드', '판매자옵션코드'에 기입
- **별도로 관리하시는 판매자용 상품/옵션 코드**

<Callout type="warning" title="주의">

1. 판매자용 상품/옵션코드로 SKU를 매칭하시는 경우, SmartShip SKU에서도 **판매자 정의코드**에 동일한 코드를 기입해야 합니다.
2. 옵션이 있는 상품은 각 **옵션**의 판매자 옵션코드에 매칭코드를 입력하세요.
3. 옵션이 없는 상품은 각 **상품**의 판매자 상품코드에 매칭코드를 입력하세요.

</Callout>

## 판매채널-SKU 연결정보 등록

판매자상품코드를 별도로 관리하지 않으시는 경우, **상품 및 재고 > 판매상품-SKU 연결** 메뉴에서 판매사이트, 상품번호, 상품명, 옵션명을 입력하고 SKU를 선택하여 맵핑 데이터를 생성할 수 있습니다.

<AnnotatedImage src="sales_item-sku-link" section="api-guide" alt="판매채널-SKU 연결정보 등록 화면" caption="상품 및 재고 > 판매상품-SKU 연결" />

해당 페이지에서는 엑셀 업로드를 이용하여 다건의 데이터를 한 번에 등록할 수 있습니다.

## SmartShip이 SKU를 찾는 과정

주문이 수집되면 SmartShip은 다음 순서로 출고할 SKU를 탐색합니다.

1. 주문 상품의 **판매자옵션/상품코드**와 일치하는 SKU no.가 있으면 해당 SKU를 선택
2. 주문 상품의 **판매자옵션코드**와 일치하는 **판매자정의코드**를 가진 SKU가 있으면 해당 SKU를 선택
3. 주문 상품의 **판매자옵션코드**와 일치하는 **바코드**가 등록된 SKU가 있으면 해당 SKU를 선택
4. 주문 상품의 **상품번호와 옵션명**으로 판매상품-SKU에 등록된 연결정보가 있으면 해당 SKU를 선택

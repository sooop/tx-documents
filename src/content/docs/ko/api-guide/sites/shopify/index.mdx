## 사전 준비 — Shopify 앱 인증

Shopify는 별도의 API 키 발급 없이 **Shopify 앱스토어에서 TracX Logis 앱을 설치**하여 인증하는 방식을 사용합니다. 스마트십은 판매자의 Shopify 아이디나 비밀번호를 저장하지 않습니다.

<Callout type="info">
Shopify 판매자 어드민에 로그인할 수 있는 계정이 준비되어 있어야 합니다. 앱 설치 시 Shopify 인증 과정을 거쳐 스마트십이 자동으로 접속 토큰을 발급받습니다.
</Callout>

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

<AnnotatedImage src="shopify-select-modal" section="api-guide" alt="스마트십 연결 쇼핑몰 선택 모달의 Shopify" caption="연결 쇼핑몰 선택에서 Shopify 클릭" />

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 쇼핑사이트 선택 목록에서 **Shopify**를 선택합니다.

### 연결 정보 입력 및 인증

<AnnotatedImage src="shopify-api-connect-form" section="api-guide" alt="스마트십 Shopify API 연결 정보 입력 화면" caption="쇼핑몰 이름 확인 후 인증 받기 버튼 클릭" />

1. 쇼핑몰 기본 연동 정보를 확인합니다.
2. 하단의 **[인증 받기]** 버튼을 클릭하면 Shopify 앱스토어 화면이 새 창으로 열립니다.

<AnnotatedImage src="shopify-app-install-page" section="api-guide" alt="Shopify 앱스토어의 TracX Logis 설치 화면" caption="Shopify 앱스토어 > Smart Ship(TracX Logis) > 설치" />

3. Shopify 앱스토어에서 **Smart Ship**(TracX Logis 개발) 앱의 **[설치]** 버튼을 클릭합니다.
4. Shopify 인증 과정을 완료하면 스마트십으로 자동 이동합니다.
5. 인증이 완료되면 접속 토큰이 저장됩니다.

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요.

### 수집 가능한 주문 상태

Shopify에서 발생한 신규 주문 중 아래 두 조건을 **모두 만족**하는 주문이 수집됩니다.

<AnnotatedImage src="shopify-payment-status" section="api-guide" alt="Shopify 주문의 Payment status = Paid" caption="Payment status 가 Paid 인 주문" />

<AnnotatedImage src="shopify-fulfillment-status" section="api-guide" alt="Shopify 주문의 Fulfillment status = Unfulfilled" caption="Fulfillment status 가 Unfulfilled 인 주문" />

| 항목 | 조건 |
|------|------|
| **Payment status** | `Paid` (결제 완료) |
| **Fulfillment status** | `Unfulfilled` (미배송) |

<Callout type="warning" title="Authorized 결제 상태란?">
**Authorized** 결제 상태는 결제 수단 제공 업체들이 구매자의 결제 수단을 인증했다는 의미이며, 결제가 완료된 상태가 **아닐 수 있습니다**.

- Shopify의 Authorized 결제는 유효기간을 가지며, 이 기간이 끝나면 `expired` / `voided` 되면서 결제가 취소됩니다.
- Authorized 상태의 결제를 완료하려면 판매자가 주문 상세 화면에서 **[Capture payment]** 버튼을 클릭해야 합니다.
- 체크아웃 시점에 자동으로 캡쳐되도록 하려면 Shopify **설정 > 지급 > 결제 매입 방법** 항목에서 변경하세요.
</Callout>

## API 키 관리

Shopify는 앱 설치를 통해 토큰을 자동으로 발급받기 때문에 별도의 API 키 만료일이 없습니다. 다만 **앱 재설치** 등의 이유로 인증을 다시 시도해야 하는 경우, 아래 절차에 따라 토큰을 갱신합니다.

### 토큰 갱신 방법

1. 메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.
2. 갱신할 Shopify 샵을 선택합니다.
3. **[토큰 갱신]** 버튼을 클릭하여 Shopify 인증을 진행합니다.
4. 최초 설정 시와 같이 Shopify에 Smart Ship 앱이 설치되어 있지 않다면 설치가 함께 진행됩니다.
5. 인증을 완료하면 접속 토큰이 갱신됩니다.
6. 하단 **[연결 테스트하기]** 버튼을 클릭하여 API 접속이 정상 작동하는지 확인합니다.

<Callout type="warning">
토큰이 갱신되지 않은 기간 동안 수집되지 않은 주문은 아래 방법으로 수동으로 가져와야 합니다.
</Callout>

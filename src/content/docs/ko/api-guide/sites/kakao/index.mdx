## 사전 준비 — 카카오 쇼핑 API 키 발급

카카오 쇼핑 서비스에 대한 API 연결 설정을 위해서는 다음의 정보가 필요합니다. 스마트십에서 API 연결을 설정하기 전에 각 플랫폼 관리도구에서 확인 및 발급을 해주세요.

| 항목 | 설명 |
|------|------|
| **샵 ID** | 톡스토어: 톡스토어 URL 마지막 ID (`store.kakao.com/{ID}`) / 선물하기: 카카오 선물하기 ID |
| **Access Key** | API 키 발급 요청 시 생성되는 고유 키 |

### API Key 발급 절차

<AnnotatedImage src="kakao-seller-menu" section="api-guide" alt="카카오 쇼핑 판매자센터 판매자정보 메뉴" caption="판매자정보 > 판매채널 정보관리 선택" />

1. 톡스토어 판매자센터 혹은 선물하기 판매자센터에 접속합니다.
2. 오른쪽 상단 **판매자 정보**를 클릭한 후, 펼쳐지는 메뉴에서 **판매채널 정보관리**를 선택하여 진입합니다.

<AnnotatedImage src="kakao-api-key-page" section="api-guide" alt="카카오 쇼핑 판매채널 정보관리 API 인증키 영역" caption="판매채널 정보관리 페이지 하단 API 인증키 영역에서 발급" />

3. 판매채널 정보관리 페이지 하단의 **API 인증키** 영역에서 **[발급하기]** 버튼을 통해 API KEY를 발급받습니다.

<Callout type="info" title="선물하기 API 권한 신청">
선물하기의 경우 API 권한을 카카오 쇼핑을 통해 발급받아 진행할 수 있습니다. 선물하기 판매자센터 내 **1:1 문의**를 통해 API 권한을 신청한 후 연동을 진행해주세요.
</Callout>

<Callout type="warning" title="API 연동대행사는 최대 3개">
카카오 쇼핑은 API 연동대행사를 최대 3개까지 등록할 수 있습니다. 이미 3개가 등록되어 있어 스마트십을 추가로 연동할 수 없는 경우, 판매자가 직접 카카오쇼핑 고객센터를 통해 기존 연동대행사와의 매핑정보 삭제를 요청해야 합니다.
</Callout>

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

<AnnotatedImage src="kakao-api-settings-new" section="api-guide" alt="스마트십 쇼핑몰 API 설정 화면" caption="쇼핑몰 API 설정 화면에서 신규등록 클릭" />

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 연결할 쇼핑사이트를 선택합니다. 이름 일부를 타이핑하면 빠르게 검색할 수 있습니다.

<AnnotatedImage src="kakao-shop-select" section="api-guide" alt="스마트십 연결 쇼핑몰 선택 - kakao" caption="검색창에 kaka 입력 후 카카오 선택" />

### 연결 정보 입력

<AnnotatedImage src="kakao-api-connect-form" section="api-guide" alt="스마트십 카카오 쇼핑 연동 등록 화면" caption="쇼핑몰 도메인, 샵 ID, API Key 입력" />

| 항목 | 입력 내용 |
|------|-----------|
| **쇼핑몰 도메인** | 톡스토어 연동: `store.kakao.com` / 선물하기 연동: `gift.kakao.com` |
| **쇼핑몰 샵 ID** | 설정하신 카카오 샵 ID (예: `store.kakao.com/{shopid}`의 `shopid`) |
| **API Key** | 발급받은 API Key |

<Callout type="info">
● 표시가 있는 필드는 필수 입력 항목입니다.
</Callout>

### 연결 테스트에 실패하는 경우

연결 테스트에 실패하는 경우 다음 사항을 확인해주세요.

1. 판매 사이트 API 키 발급 시 IP 정보가 정확하게 입력되었는지 확인합니다.
2. Access Key 등 인증 정보가 올바른 필드에 저장되었는지 확인합니다.
3. **[등록하기]** 혹은 **[수정하기]** 버튼을 클릭하여 변경 사항을 시스템에 적용했는지 확인합니다.
4. 해당 사이트의 API 서버 일시적 장애나 네트워크 문제로 연결에 실패할 수 있습니다. 잠시 후 다시 시도해보세요.

위 사항들을 확인하였음에도 접속에 실패하는 경우 `apihelp@tracxlogis.com` 으로 다음 내용을 첨부하여 문의해주세요.

- 판매자 로그인 이메일 혹은 아이디
- 연결하려는 사이트 및 샵 ID
- 스마트십의 오류 화면 캡쳐
- 쇼핑사이트 API 키 발급 시 입력한 정보 화면 캡쳐

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요. 카카오 쇼핑의 채널별 추가 옵션(예상 배송일자 업데이트, 재고 동기화 설정 등)도 해당 페이지에서 확인할 수 있습니다.
</Callout>

### 수집 가능한 주문 상태

<AnnotatedImage src="kakao-order-status" section="api-guide" alt="카카오 쇼핑 통합 주문관리 배송 요청 / 배송 준비 중 상태" caption="배송 요청 / 배송 준비 중 상태의 주문이 연동 대상" />

카카오 쇼핑에서는 자동 주문 확인 설정 여부에 따라 수집 대상 주문의 상태가 달라집니다.

- **자동 주문 확인을 사용**하는 경우: **배송 요청** 단계의 주문을 주문 확인 처리하여 수집합니다.
- **자동 주문 확인을 사용하지 않는 경우**: **배송 준비 중** 단계의 주문을 수집합니다.

취소된 주문은 수집 대상에서 제외됩니다.

<Callout type="info">
기본적으로 자동 주문 수집은 작동을 시작하는 당일부터의 주문을 수집합니다. 인증 키 만료 등의 사유로 자동 주문이 중지된 후 재설정한 경우, 주문이 수집되지 않은 기간의 주문은 반드시 수동으로 API 주문 연동을 실행해야 수집됩니다.
</Callout>

## 풀필먼트 상품 연결

풀필먼트 서비스 이용 시, 판매사이트 상품과 스마트십 SKU를 연결해야 센터에서 정확한 상품을 출고할 수 있습니다.

<AnnotatedImage src="kakao-sku-vendor-code" section="api-guide" alt="스마트십 SKU 관리 - 판매자 정의 코드 입력" caption="재고 > SKU 관리에서 판매자 정의 코드 입력" />

<ApiSkuVendorCode lang="ko" />

### 판매 사이트 상품에 SKU no. 등록

카카오 판매자센터 **상품 > 상품등록** 또는 상품 조회/수정 화면에서 상품별로 코드를 입력합니다.

| 상품 유형 | 입력 항목 | 입력 값 |
|-----------|-----------|---------|
| 단일 상품 (옵션 없음) | 판매자상품코드 | 스마트십 SKU no. 또는 판매자 정의코드 |
| 선택 상품 (옵션 있음) | **조합형** 옵션의 **관리코드** | 각 옵션의 SKU no. 또는 판매자 정의코드 |

<Callout type="warning">
**독립형** 옵션은 관리코드 입력이 불가하여 풀필먼트 주문으로 생성되지 않습니다. 풀필먼트 서비스를 이용하는 경우 반드시 **조합형** 옵션으로 상품을 등록해주세요.
</Callout>

<Callout type="info">
판매채널 상품과 SKU를 연결하는 자세한 방법은 [API 연결 후 판매사이트 설정 방법](/ko/api-guide/site-settings/)을 참고하세요.
</Callout>

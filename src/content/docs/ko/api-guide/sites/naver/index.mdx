## 사전 준비 — Naver 스마트스토어 API 대행사 등록

Naver 스마트스토어는 별도의 API 키를 직접 발급받는 방식이 아니라, 스마트스토어센터에서 **API 대행사**로 스마트십을 등록하는 방식으로 연동합니다. 등록이 완료되면 **API 연동 판매자 ID**가 자동으로 발급됩니다.

| 항목 | 설명 |
|------|------|
| **쇼핑몰 샵 ID** | 스마트스토어 샵 ID |
| **API 연동 판매자 ID** | 스마트스토어센터에서 대행사 등록 시 자동으로 발급되는 ID |

### API 대행사 등록 절차

<AnnotatedImage src="naver-api-management" section="api-guide" alt="네이버 스마트스토어센터 API 관리 화면" caption="스토어 관리 > API 관리 메뉴에서 대행사 등록" />

1. [네이버 스마트스토어센터](https://sell.smartstore.naver.com/)에 접속합니다.
2. 좌측 메뉴에서 **스토어 관리 > API 관리**로 이동합니다.
3. **API 연동 판매자 ID**가 자동으로 발급되어 표시됩니다. 스마트십 연동 시 이 값이 필요합니다.

<AnnotatedImage src="naver-api-agency-select" section="api-guide" alt="네이버 주문 API 대행사 선택 화면" caption="주문 API 대행사 목록에서 트랙스로지스-Smartship 선택" />

4. **API 대행사** 영역의 **주문 API** 항목에서 **[+ 대행사추가]** 버튼을 클릭합니다.
5. 대행사 목록에서 **트랙스로지스-Smartship**을 선택합니다.
6. 하단 **[저장]** 버튼을 눌러 설정을 저장합니다.

<Callout type="info">
API 연동 판매자 ID는 스마트스토어센터의 **API 관리** 화면 상단에서 확인할 수 있으며, 스마트십에서 API 연결을 설정할 때 이 값이 필요하므로 복사해두세요.
</Callout>

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

<AnnotatedImage src="naver-api-settings-new" section="api-guide" alt="스마트십 쇼핑몰 API 설정 화면" caption="쇼핑몰 API 설정 화면에서 신규등록 클릭" />

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 연결할 쇼핑사이트를 선택합니다. 이름 일부를 타이핑하면 빠르게 검색할 수 있습니다.

<AnnotatedImage src="naver-shop-select" section="api-guide" alt="스마트십 연결 쇼핑몰 선택 - NAVER" caption="검색창에 네 입력 후 NAVER 선택" />

### 연결 정보 입력

<AnnotatedImage src="naver-api-connect-form" section="api-guide" alt="스마트십 네이버 스마트스토어 연동 등록 화면" caption="쇼핑몰 샵 ID, API 연동 판매자 ID 입력" />

### 연결 테스트에 실패하는 경우

연결 테스트에 실패하는 경우 다음 사항을 확인해주세요.

1. 스마트스토어센터 **API 관리** 메뉴에서 **트랙스로지스-Smartship**이 주문 API 대행사로 지정되어 있고 저장되었는지 확인합니다.
2. 쇼핑몰 샵 ID와 API 연동 판매자 ID가 올바른 필드에 저장되었는지 확인합니다.
3. **[등록하기]** 혹은 **[수정하기]** 버튼을 클릭하여 변경 사항을 시스템에 적용했는지 확인합니다.
4. 해당 사이트의 API 서버 일시적 장애나 네트워크 문제로 연결에 실패할 수 있습니다. 잠시 후 다시 시도해보세요.

위 사항들을 확인하였음에도 접속에 실패하는 경우 `apihelp@tracxlogis.com` 으로 다음 내용을 첨부하여 문의해주세요.

- 판매자 로그인 이메일 혹은 아이디
- 연결하려는 사이트 및 샵 ID
- 스마트십의 오류 화면 캡쳐
- 쇼핑사이트 API 키 발급 시 입력한 정보 화면 캡쳐

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요. 네이버 스마트스토어의 채널별 추가 옵션(예상 배송일자 업데이트, 재고 동기화 설정 등)도 해당 페이지에서 확인할 수 있습니다.
</Callout>

### 수집 가능한 주문 상태

<AnnotatedImage src="naver-order-status" section="api-guide" alt="네이버 스마트스토어센터 주문/배송 현황" caption="신규주문 / 배송준비 상태의 주문이 연동 대상" />

네이버 스마트스토어에서는 **신규주문** 및 **배송준비** 상태의 주문을 수집합니다. 취소된 주문은 수집 대상에서 제외됩니다.

<Callout type="info">
기본적으로 자동 주문 수집은 작동을 시작하는 당일부터의 주문을 수집합니다. 인증 키 만료 등의 사유로 자동 주문이 중지된 후 재설정한 경우, 주문이 수집되지 않은 기간의 주문은 반드시 수동으로 API 주문 연동을 실행해야 수집됩니다.
</Callout>

### 판매 사이트 상품에 SKU no. 등록

네이버 스마트스토어센터 **상품관리 > 상품 조회/수정** 화면에서 상품별로 코드를 입력합니다.

| 상품 유형 | 입력 항목 | 입력 값 |
|-----------|-----------|---------|
| 단일 상품 (옵션 없음) | 판매자 상품코드 | 스마트십 SKU no. 또는 판매자 정의코드 |
| 선택 상품 (옵션 있음) | 각 옵션의 관리코드 | 각 옵션의 SKU no. 또는 판매자 정의코드 |

<Callout type="info">
판매채널 상품과 SKU를 연결하는 자세한 방법은 [API 연결 후 판매사이트 설정 방법](/ko/api-guide/site-settings/)을 참고하세요.
</Callout>

## 사전 준비 — 라쿠텐 API 키 발급

라쿠텐은 RMS(Rakuten Merchant Server)에서 API 키를 직접 발급받아야 합니다.

| 항목 | 설명 |
|------|------|
| **샵 ID** | 라쿠텐 RMS 로그인 시 사용하는 판매자 ID |
| **serviceSecret Key** | API 신청 후 발급되는 고유 키 |
| **License Key** | API 신청 후 함께 발급되는 고유 키 |

<Callout type="info">
serviceSecret Key와 License Key는 발급 후 **90일** 동안만 유효합니다. 만료되기 전에 동일한 절차로 재발급하고 스마트십 API 설정에서 수정해야 합니다.
</Callout>

### API 키 발급 절차

<AnnotatedImage src="rakuten-rms-api-apply-page" section="api-guide" alt="라쿠텐 RMS WEB API 서비스 신청 화면" caption="이용약관 확인 및 신청" />

1. [라쿠텐 RMS](https://glogin.rms.rakuten.co.jp/)에 로그인합니다.
2. **店舗様向け情報・サービス(판매자용 정보·서비스) > WEB APIサービス > 申込、利用規約確認(신청·이용약관 확인)** 으로 이동합니다.
3. 이용약관을 검토한 후 이용 신청을 완료합니다.
4. **WEB API** 메뉴에서 **[利用機能編集(이용 기능 편집)]** 버튼을 클릭합니다.
5. **楽天ペイ受注API(RakutenPay_OrderAPI)** 항목에서 **[全て選択(전체 선택)]** 을 클릭하여 모든 권한을 선택한 후 등록합니다.
6. 발급된 **serviceSecret Key**와 **licenseKey**를 복사해둡니다.

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 쇼핑사이트 선택 목록에서 **Rakuten(라쿠텐)** 을 선택합니다.

### 연결 정보 입력

<AnnotatedImage src="rakuten-api-connect-form" section="api-guide" alt="스마트십 라쿠텐 API 연결 정보 입력 화면" caption="샵 ID, 샵 이름, serviceSecret Key, License Key, 만료일 입력" />

| 항목 | 입력 내용 |
|------|-----------|
| **쇼핑몰 샵 ID** | 라쿠텐 RMS 로그인 판매자 ID |
| **샵 이름** | 라쿠텐 샵 URL에서 `https://www.rakuten.ne.jp/` 이후 경로 부분만 입력 (잘못 입력하면 URL 확인이 불가능하므로 정확하게 기입) |
| **serviceSecret Key** | 발급받은 serviceSecretKey |
| **License Key** | 발급받은 licenseKey |
| **API 키 만료일** | 발급 시 확인된 만료일 (만료 후 API 연동 불가) |

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요. 라쿠텐의 채널별 추가 옵션(API Key 만료일 설정, 라쿠텐 옵션코드 풀필먼트 사용 설정 등)도 해당 페이지에서 확인할 수 있습니다.
</Callout>

### 수집 가능한 주문 상태

라쿠텐에서는 **発送待ち(발송대기)** 상태의 주문이 수집됩니다.

## 풀필먼트 상품 연결

풀필먼트 서비스 이용 시, 판매사이트 상품과 스마트십 SKU를 연결해야 센터에서 정확한 상품을 출고할 수 있습니다.

### 판매 사이트 상품에 SKU no. 등록

라쿠텐 RMS **商品編集(상품 관리) > 商品一覧・登録(상품 목록·등록)** 에서 상품별로 코드를 입력합니다.

| 상품 유형 | 입력 항목 | 입력 값 |
|-----------|-----------|---------|
| 단일 상품 (옵션 없음) | 商品番号 (상품번호) | 스마트십 SKU no. 또는 판매자 정의코드 |
| 선택 상품 (옵션 있음) | システム連携用SKU番号 (시스템연계용 SKU 번호) | 각 옵션의 판매자 정의코드 |

<Callout type="info">
판매채널 상품과 SKU를 연결하는 자세한 방법은 [API 연결 후 판매사이트 설정 방법](/ko/api-guide/site-settings/)을 참고하세요.
</Callout>

## API 키 관리

라쿠텐 API Key(serviceSecret Key + License Key)는 발급 후 **90일** 동안만 유효합니다. 만료 전에 RMS에서 동일한 절차로 키를 재발급하고, 스마트십 API 설정에서 두 키를 모두 수정해야 합니다.

<Callout type="warning">
API Key가 만료되면 자동 주문 수집이 중단됩니다. 중단된 기간의 주문은 아래 방법으로 수동으로 가져와야 합니다.
</Callout>

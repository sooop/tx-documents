## 사전 준비 — 롯데온 API 키 발급

롯데온 서비스에 대한 API 연결 설정을 위해서는 다음의 정보가 필요합니다. 스마트십에서 API 연결을 설정하기 전에 롯데온 판매자 센터에서 확인 및 발급을 해주세요.

| 항목 | 설명 |
|------|------|
| **샵 ID** | 롯데온 판매자 센터에 로그인하는 판매자 ID |
| **API Key** | API 키 발급 요청 시 생성되는 고유 키 |
| **Account Number** | 롯데온에서 판매자 계정에게 발급하는 고유 번호(업체 코드) |

### API Key 발급 절차

<AnnotatedImage src="lotte-on-openapi-management" section="api-guide" alt="롯데온 판매자 센터 OpenAPI 관리 화면" caption="판매자정보 > OpenAPI 관리 메뉴에서 서버 IP 등록" />

1. [롯데온 판매자 센터](https://seller.lotteon.com/)에 접속합니다.
2. 좌측 메뉴의 **판매자정보 > OpenAPI 관리** 메뉴로 이동합니다.
3. **서버IP 등록** 영역에서 아래와 같이 설정한 후 저장합니다.

| 항목 | 입력 값 |
|------|---------|
| **연동 방법** | `호스팅/셀러툴`, `직접입력` 모두 선택 |
| **호스팅/셀러툴 선택** | `트랙스로지스` 선택 |
| **서버IP 등록** | `211.115.121.4;211.115.121.6;20.214.92.38;20.249.181.82;20.249.181.119` |

<Callout type="warning">
IP 주소를 정확히 입력하지 않으면 API 연결 테스트가 실패할 수 있습니다. 반드시 위 IP 주소를 세미콜론(`;`)으로 구분하여 정확하게 입력하세요.
</Callout>

4. 페이지 하단 **인증키 정보**에서 **인증키(API Key)**를 확인하여 저장합니다.
5. 판매자 센터에 로그인한 후 좌측 상단 스토어명 아래에 표시되는 **업체 코드(Account Number)**를 확인합니다.
6. 위에서 확인한 **인증키**와 **업체 코드**로 스마트십과의 API 연동 설정을 진행합니다.

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

<AnnotatedImage src="lotte-on-api-settings-new" section="api-guide" alt="스마트십 쇼핑몰 API 설정 화면" caption="쇼핑몰 API 설정 화면에서 신규등록 클릭" />

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 연결할 쇼핑사이트를 선택합니다. 이름 일부를 타이핑하면 빠르게 검색할 수 있습니다.

### 연결 테스트에 실패하는 경우

연결 테스트에 실패하는 경우 다음 사항을 확인해주세요.

1. 판매 사이트 API 키 발급 시 IP 정보가 정확하게 입력되었는지 확인합니다.
2. 샵 ID, API Key, Account Number 등 인증 정보가 각각 올바른 필드에 저장되었는지 확인합니다.
3. **[등록하기]** 혹은 **[수정하기]** 버튼을 클릭하여 변경 사항을 시스템에 적용했는지 확인합니다.
4. 해당 사이트의 API 서버 일시적 장애나 네트워크 문제로 연결에 실패할 수 있습니다. 잠시 후 다시 시도해보세요.

위 사항들을 확인하였음에도 접속에 실패하는 경우 `apihelp@tracxlogis.com` 으로 다음 내용을 첨부하여 문의해주세요.

- 판매자 로그인 이메일 혹은 아이디
- 연결하려는 사이트 및 샵 ID
- 스마트십의 오류 화면 캡쳐
- 쇼핑사이트 API 키 발급 시 입력한 정보 화면 캡쳐

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요. 롯데온의 채널별 추가 옵션(예상 배송일자 업데이트, 재고 동기화 설정 등)도 해당 페이지에서 확인할 수 있습니다.
</Callout>

### 수집 가능한 주문 상태

<AnnotatedImage src="lotte-on-order-status" section="api-guide" alt="롯데온 판매자 센터 주문 상태" caption="신규주문 / 상품준비 상태의 주문이 연동 대상" />

롯데온에서는 **신규주문** 및 **상품준비** 상태의 주문을 수집합니다. 취소된 주문은 수집 대상에서 제외됩니다.

<Callout type="info">
기본적으로 자동 주문 수집은 작동을 시작하는 당일부터의 주문을 수집합니다. 인증 키 만료 등의 사유로 자동 주문이 중지된 후 재설정한 경우, 주문이 수집되지 않은 기간의 주문은 반드시 수동으로 API 주문 연동을 실행해야 수집됩니다.
</Callout>

## API 키 관리

일반적으로 쇼핑 플랫폼이 발급하는 API 키에는 만료일자가 있습니다. 키가 만료되면 API 호출 권한이 사라져 주문 수집이나 송장 등록 등 정보 연동을 정상적으로 수행할 수 없습니다.

- API 키의 만료일자를 별도로 관리해주세요.
- 키가 만료되기 전에 롯데온 판매자 센터에서 API 키를 갱신 또는 재발급 받아야 합니다.
- API 키를 갱신/재발급 받은 경우 일반적으로 기존 키는 즉시 만료되므로, 스마트십의 쇼핑사이트 API 설정에서도 새 키로 수정해야 합니다.

<Callout type="warning">
API 키가 만료되면 자동 주문 수집이 중단됩니다. 설정 정보를 수정한 후에는 수집이 중단된 기간의 주문을 수동으로 API 연동해주셔야 합니다.
</Callout>

## 사전 준비 — Yahoo! Shopping API 키 발급

Yahoo! Shopping 연동을 위해서는 아래 정보가 필요합니다. 스마트십에서 API 연결을 설정하기 전에 Yahoo! Shopping 관리 도구에서 미리 확인·발급해두세요.

| 항목 | 설명 |
|------|------|
| **쇼핑몰 샵 ID** | Yahoo! JAPAN 비즈니스 ID |
| **Store Account ID** | Yahoo! ストアアカウント(스토어 계정) |
| **Public Key** | Yahoo! Shopping 공개키 |
| **Public Key Version** | Yahoo! Shopping 공개키 버전 |

<Callout type="warning">
공개키를 입력하지 않으면 **하루에 한 번** 토큰을 재발행하여 인증해야 합니다. 공개키를 등록하면 인증 유효 시간을 **한 달에 한 번**으로 늘릴 수 있습니다.
</Callout>

### 비즈니스 ID 및 Store Account ID 확인

<AnnotatedImage src="yahoo-jp-business-id" section="api-guide" alt="Yahoo! JAPAN 비즈니스 ID 확인 화면" caption="従業員情報 > ID登録情報에서 Yahoo! JAPANビジネスID 확인" />

Yahoo! JAPAN 비즈니스 관리 도구 **従業員情報 > ID登録情報** 에서 **Yahoo! JAPANビジネスID**(비즈니스 ID)를 확인합니다.

<AnnotatedImage src="yahoo-jp-store-account" section="api-guide" alt="Yahoo! ストアアカウント(Store Account) 확인 화면" caption="ストアクリエイターPro 상단에서 ストアアカウント 확인" />

**ストアクリエイターPro(Store Creator Pro)** 좌측 상단에서 **ストアアカウント**(Store Account ID)를 확인합니다.

### 공개키 발행

<AnnotatedImage src="yahoo-jp-public-key-page" section="api-guide" alt="Yahoo! Shopping 暗号鍵管理(암호키 관리) 화면" caption="設定 > 暗号鍵管理 > 公開鍵発行에서 공개키 발행" />

1. **設定(설정) > 暗号鍵管理(암호키 관리)** 메뉴로 이동합니다.
2. **[公開鍵発行(공개키 발행)]** 버튼을 클릭하여 공개키를 발행합니다.
3. 발행된 **公開鍵(공개키)** 오른쪽의 **[コピー(복사)]** 버튼을 클릭하여 값을 복사합니다.
4. **バージョン(버전)** 값도 별도로 확인해둡니다.

<Callout type="warning">
- 공개키는 발행일로부터 **1년간** 유효합니다. **[有効日(유효일)]** 을 기준으로 만료 전에 새 공개키를 발급받아 교체해야 합니다.
- 공개키 만료 전 Yahoo에서 메일이 발송되므로, 메일 수신 시 공개키를 새로 발급하여 변경해 주세요.
- Public Key는 복사한 내용을 **그대로** 스마트십에 붙여넣어야 합니다. 앞뒤 공백이나 개행이 변경되지 않도록 주의하세요.
</Callout>

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 쇼핑사이트 선택 목록에서 **Yahoo! Shopping** 또는 **Yahoo JP**를 선택합니다.

### 연결 정보 입력 및 인증

<AnnotatedImage src="yahoo-jp-api-connect-form" section="api-guide" alt="스마트십 Yahoo! Shopping API 연결 정보 입력 화면" caption="쇼핑몰 샵 ID, Store Account ID, 공개키, 공개키 버전 입력" />

| 항목 | 입력 내용 |
|------|-----------|
| **쇼핑몰 샵 ID** | Yahoo! JAPAN 비즈니스 ID |
| **Store Account ID** | Yahoo! ストアアカウント(스토어 계정) |
| **공개키** | Yahoo에서 발행한 Public Key (복사한 내용 그대로 붙여넣기) |
| **공개키 버전** | Public Key Version |

1. 위 항목을 입력한 후 하단의 **[인증 받기]** 버튼을 클릭합니다.
2. 새 창으로 열리는 Yahoo! Shopping 인증 페이지에서 야후 아이디와 비밀번호를 입력하여 로그인합니다.
3. 판매자 휴대폰 번호로 전송된 인증 코드를 입력하여 인증을 완료합니다.
4. 인증이 완료되면 창이 닫히고 토큰이 저장됩니다.

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요.
</Callout>

### 수집 가능한 주문 상태

Yahoo! Shopping에서는 **결제 완료된 신규 주문** 및 판매자 확인 처리된 **배송 준비** 상태의 주문이 수집됩니다. 취소된 주문은 수집 대상에서 제외됩니다.

### 판매 사이트 상품에 SKU no. 등록

<AnnotatedImage src="yahoo-jp-product-sku-input" section="api-guide" alt="Yahoo! Shopping 상품 코드 입력 화면" caption="商品管理 > 商品の照会/修正에서 상품 코드 입력" />

Yahoo! Shopping **商品管理(상품관리) > 商品の照会/修正(상품조회/수정)** 에서 상품별로 코드를 입력합니다.

| 상품 유형 | 입력 항목 | 입력 값 |

| 선택 상품 (옵션 있음) | サブコード (서브 코드) | 각 옵션의 판매자 정의코드 |

<Callout type="info">
판매채널 상품과 SKU를 연결하는 자세한 방법은 [API 연결 후 판매사이트 설정 방법](/ko/api-guide/site-settings/)을 참고하세요.
</Callout>

## API 키 관리

Yahoo! Shopping의 Public Key는 발급 후 **1년** 동안 유효합니다. 만료 전에 Yahoo! Shopping 관리 도구에서 동일한 절차로 공개키를 재발급하고, 스마트십 API 설정에서 공개키와 버전을 모두 수정해야 합니다.

<Callout type="warning">
Public Key가 만료되면 자동 주문 수집이 중단됩니다. 중단된 기간의 주문은 아래 방법으로 수동으로 가져와야 합니다.
</Callout>

## 사전 준비 — TikTok Shop 샵 ID 확인

TikTok Shop은 별도의 API 키를 발급받을 필요 없이, 스마트십에서 **샵 ID**를 입력하고 TikTok Shop 판매자 계정으로 인증하면 접속 토큰이 자동으로 발급됩니다.

| 항목 | 설명 |
|------|------|
| **샵 ID** | TikTok Shop Seller Center에서 사용하는 샵 식별자 |
| **접속 토큰** | 스마트십에 샵 ID를 등록한 후 TikTok Shop 인증 과정을 통해 자동 발급 |

<Callout type="info">
쇼핑 사이트의 판매자 어드민을 사용하기 위해서 판매자 ID와 패스워드로 로그인하는 것과 마찬가지로, API를 사용하여 주문 정보를 처리하기 위해서는 쇼핑사이트가 제공하는 인증 정보(접속 토큰)가 필요합니다. 접속 토큰은 쇼핑 사이트의 로그인 과정을 거쳐 원격으로 발급되며, 스마트십은 판매자의 쇼핑 사이트 아이디나 패스워드를 획득하거나 저장하지 않습니다.
</Callout>

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 상단의 **[신규등록]** 버튼을 클릭합니다.
2. 쇼핑사이트 선택 목록에서 **TikTok Shop**을 선택합니다. (검색창에 사이트 이름의 일부를 입력하면 빠르게 찾을 수 있습니다.)

### 연결 정보 입력

<AnnotatedImage src="tiktok-api-connect-form" section="api-guide" alt="스마트십 TikTok Shop API 연결 정보 입력 화면" caption="쇼핑몰 샵 ID 입력 후 [인증받기] 클릭" />

1. **쇼핑몰 샵 ID** 란에 샵 ID 값을 입력합니다.
2. 하단 **[인증받기]** 버튼을 클릭합니다.
3. `services.tiktokshop.com` 주소의 팝업창이 열리면, **서비스 국가를 선택**한 뒤 로그인 아이디와 비밀번호를 입력하고 **[Submit]** 버튼을 클릭합니다.
4. 팝업창이 닫히면서 인증 정보가 자동으로 저장됩니다.

## API 키 관리

TikTok Shop의 접속 토큰에는 만료일자가 있습니다. 토큰이 만료되면 API 호출 권한이 사라져 주문을 수집하거나 송장을 등록하는 등 정보 연동이 정상적으로 수행되지 않습니다. 만료 전에 스마트십의 쇼핑사이트 API 설정에서 토큰을 갱신해주세요.

<Callout type="warning">
토큰 갱신 후에는, 주문 수집이 중단된 기간의 주문을 반드시 수동으로 API 주문 연동하여 수집해야 합니다.
</Callout>

### TikTok Shop API 키 만료 시 갱신 방법

<AnnotatedImage src="tiktok-token-renewal" section="api-guide" alt="스마트십 TikTok Shop 토큰 갱신 화면" caption="인증 정보란의 [토큰 갱신] 버튼 클릭" />

메뉴 **주문 > 쇼핑몰 연동 > 쇼핑몰 API 설정**으로 이동합니다.

1. 키를 변경할 TikTok Shop 샵을 선택합니다.
2. 인증 정보란에 있는 **[토큰 갱신]** 버튼을 클릭합니다.
3. 초기 설정 때와 동일하게 TikTok Shop 인증 창이 열립니다. Seller Center 로그인 아이디와 비밀번호를 입력하여 판매자 계정을 인증합니다.
4. 인증에 성공하면 자동으로 키가 갱신됩니다.
5. 하단의 **[연결 테스트하기]** 버튼을 클릭하여 API 통신이 정상적으로 이루어지는지 확인합니다.
6. 토큰이 만료되어 주문이 수집되지 않은 기간의 주문이 누락되지 않도록, API 주문 수집을 수동으로 실행합니다.

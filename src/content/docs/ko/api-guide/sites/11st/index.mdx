## 사전 준비 — 11번가 API 키 발급

11번가 서비스에 대한 API 연결 설정을 위해서는 다음의 정보가 필요합니다. 스마트십에서 API 연결을 설정하기 전에 11번가 셀러오피스에서 확인 및 발급을 해주세요.

| 항목 | 설명 |
|------|------|
| **샵 ID** | 11번가 셀러오피스 판매자 ID |
| **API Key** | 11번가 OPEN API CENTER에서 발급 |

### API Key 발급 절차

<AnnotatedImage src="11st-openapi-service-register" section="api-guide" alt="11번가 OPEN API CENTER 사용자 정보 입력 화면" caption="서비스 등록/확인 진입 후 사용자 정보 입력" />

1. [11번가 OPEN API CENTER](https://openapi.11st.co.kr/)에 접속한 후 **서비스 등록/확인** 메뉴를 선택하여 진입합니다.
2. 사용자 정보를 입력한 후 **[확인]** 버튼을 클릭합니다.

<AnnotatedImage src="11st-seller-api-info" section="api-guide" alt="11번가 OPEN API CENTER Seller API 정보 영역" caption="Seller API 정보 영역의 [수정] 버튼 클릭" />

3. **Seller API 정보** 영역에서 **[수정]** 버튼을 클릭합니다.

<AnnotatedImage src="11st-seller-api-form" section="api-guide" alt="11번가 Seller API 정보 수정 화면" caption="호스팅 여부, IP 정보, 11번가 담당 MD 입력" />

4. Seller API 정보를 다음과 같이 설정한 후 **[수정]** 버튼을 클릭합니다.

| 항목 | 입력 값 |
|------|---------|
| **호스팅 여부** | `직접입력` 선택 |
| **개발서버 IP** | `20.249.181.82;20.249.181.119;20.214.92.38;211.115.121.4;211.115.121.6` |
| **개발자 PC** | `20.249.181.82;20.249.181.119;20.214.92.38;211.115.121.4;211.115.121.6` |
| **상용서버 IP** | `20.249.181.82;20.249.181.119;20.214.92.38;211.115.121.4;211.115.121.6` |
| **11번가 담당 MD** | 담당 MD가 없는 경우 `없음` 입력 |

<Callout type="warning">
IP 주소를 정확히 입력하지 않으면 API 연결 테스트가 실패할 수 있습니다. 반드시 위 IP 주소를 세미콜론(`;`)으로 구분하여 정확하게 입력하세요.
</Callout>

5. 발급된 **11ST OPEN API KEY**를 확인하여 저장합니다. 스마트십 연결 시 이 값이 필요합니다.

## 스마트십에서 API 연결 추가

### 신규 API 연결 추가

<AnnotatedImage src="11st-api-settings-new" section="api-guide" alt="스마트십 쇼핑몰 API 설정 화면" caption="쇼핑몰 API 설정 화면에서 신규등록 클릭" />

2. 연결할 쇼핑사이트를 선택합니다. 이름 일부를 타이핑하면 빠르게 검색할 수 있습니다.

### 연결 테스트에 실패하는 경우

연결 테스트에 실패하는 경우 다음 사항을 확인해주세요.

1. 판매 사이트 API 키 발급 시 IP 정보가 정확하게 입력되었는지 확인합니다.
2. 샵 ID와 API Key 등 인증 정보가 각각 올바른 필드에 저장되었는지 확인합니다.
3. **[등록하기]** 혹은 **[수정하기]** 버튼을 클릭하여 변경 사항을 시스템에 적용했는지 확인합니다.
4. 해당 사이트의 API 서버 일시적 장애나 네트워크 문제로 연결에 실패할 수 있습니다. 잠시 후 다시 시도해보세요.

위 사항들을 확인하였음에도 접속에 실패하는 경우 `apihelp@tracxlogis.com` 으로 다음 내용을 첨부하여 문의해주세요.

- 판매자 로그인 이메일 혹은 아이디
- 연결하려는 사이트 및 샵 ID
- 스마트십의 오류 화면 캡쳐
- 쇼핑사이트 API 키 발급 시 입력한 정보 화면 캡쳐

## API 설정 및 연동 주문

API 연동 동작(자동 주문 수집, 자동 주문 확인, 풀필먼트 출하지시 등)을 조절하는 옵션은 **[쇼핑몰 API 설정 > 주문 처리 설정]** 에서 변경할 수 있습니다.

<Callout type="tip">
옵션별 상세 설명은 [API 연동 시 주문 처리 옵션 가이드](/ko/api-guide/order-options/)를 참고하세요. 11번가의 채널별 추가 옵션(예상 배송일자 업데이트, 재고 동기화 설정 등)도 해당 페이지에서 확인할 수 있습니다.
</Callout>

### 수집 가능한 주문 상태

<AnnotatedImage src="11st-order-status" section="api-guide" alt="11번가 셀러오피스 배송관리 화면 결제완료 / 배송준비중 상태" caption="결제완료 / 배송준비중 상태의 주문이 연동 대상" />

11번가에서는 **결제완료** 및 **배송준비중** 상태의 주문을 수집합니다. 취소된 주문은 수집 대상에서 제외됩니다.

<Callout type="info">
기본적으로 자동 주문 수집은 작동을 시작하는 당일부터의 주문을 수집합니다. 인증 키 만료 등의 사유로 자동 주문이 중지된 후 재설정한 경우, 주문이 수집되지 않은 기간의 주문은 반드시 수동으로 API 주문 연동을 실행해야 수집됩니다.
</Callout>

### 판매 사이트 상품에 SKU no. 등록

11번가 셀러오피스 **상품관리 > 상품등록** 또는 **상품 조회/수정** 화면에서 상품별로 코드를 입력합니다.

| 상품 유형 | 입력 항목 | 입력 값 |
|-----------|-----------|---------|
| 단일 상품 (옵션 없음) | 판매자상품코드 | 스마트십 SKU no. 또는 판매자 정의코드 |
| 선택 상품 (옵션 있음) | 셀러재고코드 | 각 옵션의 SKU no. 또는 판매자 정의코드 |

<Callout type="info">
판매채널 상품과 SKU를 연결하는 자세한 방법은 [API 연결 후 판매사이트 설정 방법](/ko/api-guide/site-settings/)을 참고하세요.
</Callout>

## API 키 관리

일반적으로 쇼핑 플랫폼이 발급하는 API 키에는 만료일자가 있습니다. 키가 만료되면 API 호출 권한이 사라져 주문 수집이나 송장 등록 등 정보 연동을 정상적으로 수행할 수 없습니다.

- API 키의 만료일자를 별도로 관리해주세요.
- 키가 만료되기 전에 11번가 OPEN API CENTER에서 API 키를 갱신 또는 재발급 받아야 합니다.
- API 키를 갱신/재발급 받은 경우 일반적으로 기존 키는 즉시 만료되므로, 스마트십의 쇼핑사이트 API 설정에서도 새 키로 수정해야 합니다.

<Callout type="warning">
API 키가 만료되면 자동 주문 수집이 중단됩니다. 설정 정보를 수정한 후에는 수집이 중단된 기간의 주문을 수동으로 API 연동해주셔야 합니다.
</Callout>

## Q1: 주문 연동이 되지 않아요!

우선 쇼핑사이트의 주문 상태를 확인해주세요. 송장이 등록되었거나 이미 '배송중' 상태로 변경하셨다면, SmartShip에서는 배송처리 대상이 아닌 것으로 인식합니다.

일시적인 네트워크 이상이나 판매사이트 점검 등으로 API 서비스가 중단되었던 시간에는 주문 자동 수집이 되지 않았을 수 있습니다. 이 경우 수동으로 **주문 가져오기**를 실행해주세요.

<Collapse title="주문 가져오기 방법">

메뉴: **주문 > 쇼핑몰 연동 > 쇼핑몰 연동신청**

<AnnotatedImage src="manual-api-import" section="api-guide" alt="주문 가져오기 화면" caption="주문 > 쇼핑몰 연동 > 쇼핑몰 연동신청" />

1. 연동할 판매채널을 선택합니다.
2. 연동일 범위를 확인하세요. 수집할 대상이 결제된 시점을 포함하도록 기간을 설정합니다.
3. **주문 가져오기** 버튼을 클릭합니다.

</Collapse>

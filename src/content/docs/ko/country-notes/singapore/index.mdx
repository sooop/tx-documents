싱가포르 세관의 수입 정책 및 관련 주의사항을 정리한 가이드입니다. 아래 내용은 참고용으로만 활용해 주세요. 수입 제한 품목 및 관세 조건은 세관 정책에 따라 변경되거나 다르게 적용될 수 있습니다.

## 발송 금지 품목

싱가포르로 발송이 불가하거나 제한되는 주요 품목은 아래와 같습니다.

| 분류 | 품목 |
|------|------|
| 식품류 | 건어물, 육포, 껌, 주류, 소시지, 농수산물 및 냉동식품, 부패성이 높은 식품 |
| 의약품 | 전문 의약품, 처방약 |
| 전자기기 | 보조 배터리, 블루투스 스피커 |
| 생필품 | 에탄올 포함 제품 (디퓨저, 손세정제, 바디미스트, 섬유향수 등), 알코올 포함 제품, 캔 타입 스프레이 (인화성), 매니큐어 |
| 기타 | 담배, 씨앗, 순은, 화폐, 카드, 개인정보 서류, 총기 모양의 장난감 |

<Callout type="warning" title="액체류 제한">
액체류는 1갤런(약 3.7L)으로 발송 수량이 제한됩니다.
</Callout>

<Callout type="info" title="Express(EXP) 발송 주의">
Express(EXP) 서비스는 우체국 대행 발송입니다. 국제특급(EMS) 발송 조건을 별도로 확인한 후 발송해 주세요.
</Callout>

## 규격 제한

| 항목 | 기준 |
|------|------|
| 최대 무게 | 30kg |
| 최장변 | 150cm 이하 |
| 둘레 합계 | 가로 + (세로 + 높이) × 2 ≤ 300cm |
| 보상 한도 | USD 100 |

### 부피 무게 계산

```
부피 무게(kg) = 가로(cm) × 세로(cm) × 높이(cm) ÷ 6,000
```

부피 무게와 실제 중량 중 더 큰 값을 기준으로 요금이 책정됩니다.

## GST(부가세) 기준

싱가포르는 일정 기준을 초과하는 수입 물품에 대해 GST(Goods and Services Tax, 부가가치세)를 부과합니다.

| 항목 | 내용 |
|------|------|
| 과세 기준 | 신고가 SGD 400 초과 시 적용 |
| 세율 | 9% (2024년 1월 1일부터) |
| 통관 수수료 | SGD 12.00 (건당) |

### 계산 예시

신고가 SGD 380 + 배송비 SGD 70인 경우:

- **과세 표준**: SGD 380 + SGD 70 = **SGD 450** (SGD 400 초과이므로 GST 적용)
- **GST**: SGD 450 × 9% = **SGD 40.5**
- **통관 수수료**: SGD 12.00
- **총 납부액**: SGD 40.5 + SGD 12.00 = **SGD 52.5**

<Callout type="danger" title="통화 단위 오기재 주의">
통화 단위를 잘못 기재하여 과청구가 발생한 경우 사후 정정이 불가합니다. 신고 시 반드시 정확한 통화 단위(SGD)로 입력해 주세요.
</Callout>

<Callout type="warning" title="LVG GST 중복 납부 가능성">
LVG(Low Value Goods) GST는 판매처에서 이미 징수한 경우에도 통관 시 별도로 부과될 수 있습니다. 판매처의 세금 정책과 무관하게 중복 납부가 발생할 수 있으니 유의해 주세요.
</Callout>

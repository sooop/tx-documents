입고된 재고를 기반으로 풀필먼트 출고 주문을 등록합니다.

<Callout type="info" title="메뉴 경로">
주문 → [주문 관리](https://smartship.tracxlogis.com/Order)
</Callout>

## 합포장 기준

**참조 주문번호 = 수취인명 = 연락처(HP)**

<Callout type="tip" title="참조 주문번호란?">
각 판매 사이트에서 제공하는 합포장 기준값을 입력합니다. 장바구니 번호, 카트번호, 배송번호 등이 이에 해당합니다.
</Callout>

---

## 1. 개별 등록

주문건을 한 건씩 직접 입력하여 등록하는 방식입니다.

<AnnotatedImage src="register-order-individual" section="fulfillment" alt="풀필먼트 주문 개별 등록 화면" caption="주문 관리 > 개별 배송 등록하기 화면" />

1. 주문 관리 화면 우측 상단에서 **[개별 배송 등록하기]** 를 선택합니다.
2. 주문 정보를 입력합니다. **[출고 타입 선택]** 에서 **QFC 출고** 를 선택하세요.
3. **[새로운 상품 추가]** 를 클릭합니다.
4. SKU를 검색합니다.
5. 검색된 SKU를 선택한 후 **[완료]** 를 클릭합니다.
6. **[주문 등록하기]** 를 클릭하여 등록을 완료합니다.

---

## 2. 엑셀 등록

여러 주문건을 엑셀 파일로 한 번에 업로드하는 방식입니다.

### 2-1. 표준 엑셀 양식

SmartShip에서 제공하는 기본 양식을 사용합니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[표준 엑셀 양식]** 을 선택합니다.
3. **[포맷 다운로드]** 를 클릭하여 양식을 다운로드합니다.
4. 필요한 항목을 입력합니다. **SKU 번호 입력이 필수** 입니다.
5. **[스마트십 엑셀 파일]** 을 선택하고 **[파일 선택하기]** 로 파일을 업로드합니다.
6. 업로드 결과를 확인합니다.
7. **[주문 등록]** 을 클릭합니다.

### 2-2. 주요 쇼핑몰 엑셀 양식

쇼핑몰에서 내보낸 엑셀 파일을 그대로 업로드할 수 있습니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[주요 쇼핑몰 엑셀양식]** 을 선택합니다.
3. **[사이트 선택]** 후 파일을 업로드합니다.
4. 업로드 결과를 확인합니다.
5. **[주문 등록]** 을 클릭합니다.

### 2-3. 나만의 엑셀 양식

원하는 엑셀 양식을 SmartShip 시스템 컬럼에 매핑하여 사용하는 방식입니다. 한 번 설정 후 반복 사용이 가능합니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[나만의 엑셀양식 사용]** 을 선택합니다.
3. **[나의 엑셀 양식 편집]** 을 클릭합니다.
4. **[엑셀 업로드]** 로 나의 엑셀 파일을 업로드합니다.
5. 나의 엑셀 컬럼과 시스템 컬럼을 매핑합니다. (필수 값은 반드시 매핑)
6. 매핑 이름을 입력한 후 **[저장]** 을 클릭합니다.
7. 엑셀 양식을 선택하고 파일을 업로드합니다.
8. **[주문 등록]** 을 클릭합니다.

---

## 3. API 연동 등록

API 연동 시 별도 동작 없이 주문건이 자동으로 등록됩니다.

API 설정 방법은 [API 연동 가이드](/ko/api-guide/)를 참조하세요.

---

## 주문 등록 확인

등록된 주문건은 주문 관리 화면의 **[상태별 집계]** 에서 확인할 수 있습니다.

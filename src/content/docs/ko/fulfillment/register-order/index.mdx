입고된 재고를 기반으로 풀필먼트 출고 주문을 등록합니다.

<Callout type="info" title="메뉴 경로">
주문 → [주문 관리](https://smartship.tracxlogis.com/Order)
</Callout>

## 합포장 기준

**참조 주문번호 = 수취인명 = 연락처(HP)**

<Callout type="tip" title="참조 주문번호란?">
각 판매 사이트에서 제공하는 합포장 기준값을 입력합니다. 장바구니 번호, 카트번호, 배송번호 등이 이에 해당합니다.
</Callout>

---

## 1. 개별 등록

주문건을 한 건씩 직접 입력하여 등록하는 방식입니다. 주문 관리 화면 우측 상단에서 **[개별 배송 등록하기]** 를 선택하면 아래와 같은 **주문 생성** 창이 열립니다.

<AnnotatedImage src="register-order-form" section="fulfillment" alt="풀필먼트 주문 생성 화면 전체" caption="주문 생성 창 — 기본 정보 · 배송 설정 · 수취인 정보 · 주문 상품 영역으로 구성됩니다" />

### 1-1. 기본 정보 입력

<AnnotatedImage src="register-order-basic-info" section="fulfillment" alt="기본 정보 입력 영역" caption="기본 정보 영역" />

1. **서비스 타입** — 출고 타입을 선택합니다. 풀필먼트 출고는 **풀필먼트** 를 선택합니다.
2. **창고** — 출고할 창고를 선택합니다. **Auto** 로 두면 재고가 있는 창고에서 자동으로 출고됩니다.
3. **발송인 정보** — 기본 발송지가 자동으로 지정됩니다. 변경이 필요하면 **[주소록]** 에서 다른 발송지를 선택합니다.

### 1-2. 배송 설정

1. **배송 등급** — Standard / Express 등 배송 등급을 선택합니다.
2. **참조 주문번호** — 합포장 기준이 되는 주문번호(장바구니·카트 번호 등)를 입력합니다.
3. **주문 발생 플랫폼** — 주문이 발생한 판매 채널을 선택합니다.
4. **예약 출고일** — 특정 날짜에 출고를 예약하려는 경우 날짜를 지정합니다.
5. **비고** — 출고 시 참고할 메모를 자유롭게 입력합니다.

### 1-3. 수취인 정보 입력

배송 국가에 맞춰 입력 항목이 달라집니다. 우편번호 자동 검색 기능을 활용하면 주소 입력 시간을 크게 줄일 수 있습니다.

<Callout type="tip" title="우편번호 자동 검색">
**우편번호 입력란에 올바른 자릿수만큼 입력하면 자동으로 주소가 검색됩니다.** 별도로 검색 버튼을 누르지 않아도 됩니다.

- 해당 우편번호에 연결된 **주소가 1개뿐이면 자동으로 주소 칸이 채워집니다.**
- 여러 후보가 있는 경우 **드롭다운 목록에서 선택**하면 바로 입력됩니다.
- 이후 **상세 주소(건물명·호수 등)만 직접 입력**하면 됩니다.
</Callout>

**대한민국 — 우편번호 1개만 있는 경우 (자동 입력)**

<AnnotatedImage src="register-order-recipient-kr" section="fulfillment" alt="대한민국 수취인 정보 입력 — 우편번호 자동 입력 예시" caption="우편번호 `06984` 입력 시 주소가 즉시 자동으로 채워진 화면" />

**일본 — 여러 주소 후보가 있는 경우 (드롭다운 선택)**

<AnnotatedImage src="register-order-recipient-jp" section="fulfillment" alt="일본 수취인 정보 입력 — 우편번호 드롭다운 예시" caption="우편번호를 입력하면 후보 주소가 드롭다운으로 나타나며, 원하는 항목을 선택하면 주소가 채워집니다" />

### 1-4. 주문 상품 추가

주문 상품 영역의 **[상품 추가]** 를 클릭하면 **SKU 선택하기** 창이 열립니다.

<AnnotatedImage src="register-order-sku-selector" section="fulfillment" alt="SKU 선택 모달" caption="SKU 선택하기 창 — 창이 열리면 전체 SKU 목록이 자동으로 조회됩니다" />

1. 창이 열리면 **전체 상품이 자동으로 조회**됩니다. 대부분의 경우 재검색 없이 바로 원하는 상품을 찾을 수 있습니다.
2. 원하는 SKU를 목록에서 선택한 뒤 **[완료]** 를 클릭합니다.

<Callout type="tip" title="재검색 대신 그리드 필터를 활용하세요">
상단의 **검색 조건 / 검색어**로 다시 검색하지 않아도, **표 우측 상단의 필터 입력란**에 원하는 문자열을 입력하면 현재 조회된 목록에서 즉시 걸러집니다.

- **SKU 번호**, **판매자 정의 코드**, **상품명**, **옵션명**의 **일부분**만 입력해도 즉시 필터링됩니다.
- 예: `7547` 만 입력해도 해당 숫자가 포함된 SKU가 하이라이트되어 나타납니다.
- 재검색 시 발생하는 페이지 이동·로딩 시간이 없어, 여러 상품을 연속으로 찾을 때 매우 빠릅니다.
</Callout>

SKU를 선택하면 **배송상품 상세정보** 창이 열립니다. 필요한 항목을 보완한 뒤 **[적용]** 을 클릭합니다.

<AnnotatedImage src="register-order-product-detail" section="fulfillment" alt="배송상품 상세정보 입력 창" caption="배송상품 상세정보 — 수량·단가·원산지·HS Code 등을 필요에 따라 입력합니다" />

- **수량 / 통화 / 단가** — 기본값이 입력되어 있으며, 필요 시 수정합니다.
- **판매채널 주문정보** — 판매사이트 장바구니·주문 번호, 상품 URL을 입력하면 추적에 도움이 됩니다.
- **상품 속성** — 원산지·상품 소재·HS(HSN) Code는 해외 배송 시 통관에 활용됩니다.

### 1-5. 주문 등록 완료

모든 정보를 입력한 뒤 우측 하단의 **[등록]** 버튼을 클릭하면 주문이 생성되고, 패킹 번호가 발급됩니다.

<AnnotatedImage src="register-order-done" section="fulfillment" alt="주문 등록 완료 알림" caption="등록이 완료되면 패킹 번호가 표시됩니다" />

## 2. 엑셀 등록

여러 주문건을 엑셀 파일로 한 번에 업로드하는 방식입니다.

### 2-1. 표준 엑셀 양식

SmartShip에서 제공하는 기본 양식을 사용합니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[표준 엑셀 양식]** 을 선택합니다.
3. **[포맷 다운로드]** 를 클릭하여 양식을 다운로드합니다.
4. 필요한 항목을 입력합니다. **SKU 번호 입력이 필수** 입니다.
5. **[스마트십 엑셀 파일]** 을 선택하고 **[파일 선택하기]** 로 파일을 업로드합니다.
6. 업로드 결과를 확인합니다.
7. **[주문 등록]** 을 클릭합니다.

### 2-2. 주요 쇼핑몰 엑셀 양식

쇼핑몰에서 내보낸 엑셀 파일을 그대로 업로드할 수 있습니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[주요 쇼핑몰 엑셀양식]** 을 선택합니다.
3. **[사이트 선택]** 후 파일을 업로드합니다.
4. 업로드 결과를 확인합니다.
5. **[주문 등록]** 을 클릭합니다.

### 2-3. 나만의 엑셀 양식

원하는 엑셀 양식을 SmartShip 시스템 컬럼에 매핑하여 사용하는 방식입니다. 한 번 설정 후 반복 사용이 가능합니다.

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[나만의 엑셀양식 사용]** 을 선택합니다.
3. **[나의 엑셀 양식 편집]** 을 클릭합니다.
4. **[엑셀 업로드]** 로 나의 엑셀 파일을 업로드합니다.
5. 나의 엑셀 컬럼과 시스템 컬럼을 매핑합니다. (필수 값은 반드시 매핑)
6. 매핑 이름을 입력한 후 **[저장]** 을 클릭합니다.
7. 엑셀 양식을 선택하고 파일을 업로드합니다.
8. **[주문 등록]** 을 클릭합니다.

## 3. API 연동 등록

API 연동 시 별도 동작 없이 주문건이 자동으로 등록됩니다.

API 설정 방법은 [API 연동 가이드](/ko/api-guide/)를 참조하세요.

등록된 주문건은 주문 관리 화면의 **[상태별 집계]** 에서 확인할 수 있습니다.

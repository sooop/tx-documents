풀필먼트 센터에 보관 중인 재고를 직접 반출할 수 있습니다.

<Callout type="info" title="메뉴 경로">
상품 및 재고 → [재고 출고 신청](https://smartship.tracxlogis.com/Inventory/RTV)
</Callout>

<Callout type="tip" title="소량 발송 안내">
소량 물량의 경우 **[개별 등록]** 을 통해 택배 발송으로 신청하는 것을 권장합니다. 해당 메뉴를 통한 출고 시에는 사전에 센터와 일정을 조율한 후 직접 용차 신청을 진행해 주세요. (별도 용차가 필요한 경우 센터에 문의)
</Callout>

## 재고 반출 요청 방법

<AnnotatedImage src="request-export-page" section="fulfillment" alt="재고 출고 신청 화면" caption="상품 및 재고 > 재고 출고 신청 화면" />

### ① SKU 추가하기

**[SKU 추가하기]** 버튼을 클릭하여 반출할 SKU를 추가합니다.

### ② 발송 주소 입력

도착 국가 및 수령인 주소 정보를 입력합니다.

### ③ 등록하기

**[등록하기]** 버튼을 클릭하여 반출 신청을 완료합니다.

</Callout>

## 재고 현황

<AnnotatedImage src="check-inventory" section="fulfillment" alt="재고 현황 화면" caption="상품 및 재고 > 재고 현황 화면" />

### ① 국가·창고별 재고

국가 및 창고별 총 재고 수량을 확인할 수 있습니다.

### ② SKU별 재고 수량

각 SKU별 세부 재고 현황을 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **창고 보유 수량** | 현재 창고에 보유하고 있는 재고 수량 |
| **가용 재고** | 주문 할당을 제외하고 실제 사용 가능한 재고 수량 |
| **주문 할당 수량** | 주문이 접수되어 할당된 재고 수량 |
| **만료/파손됨** | 유통기한 만료 또는 파손된 재고 수량 |
| **입고 예정** | 입고 신청되어 센터 도착 예정인 재고 수량 |

### ③ SKU별 입출고 상세 내역

SKU별 입고·출고 일자별 상세 내역을 확인할 수 있습니다.

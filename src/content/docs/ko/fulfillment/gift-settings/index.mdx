title: "풀필먼트 사은품 설정"
description: "풀필먼트 주문에 사은품을 자동으로 추가하는 정책 설정 방법을 안내합니다."
---

등록된 주문건에 자동으로 사은품을 추가하는 기능입니다. 사은품 적용 설정 및 세부 정책 설정을 통해 조건에 맞는 주문에 사은품을 지급할 수 있습니다.

## 메뉴 접근 경로

**주문 → 사은품 정책 관리**

## 사은품 자동 적용 설정

<AnnotatedImage src="gift-policy-auto-settings" section="fulfillment" alt="사은품 자동 적용 설정 화면" caption="사은품 정책 관리 > 자동 적용 설정" />

① **정책 적용 방식** — [자동 적용] 또는 [수동 적용] 중 선택합니다.

② **출하 지시 설정(풀필먼트 주문)** — 정책 적용 방식이 [자동 적용]인 경우, 사은품 적용 후 출하 지시를 자동으로 진행하도록 설정할 수 있습니다.

③ **사은품 중복 적용** — 여러 정책의 조건을 동시에 충족하는 경우 처리 방식을 선택합니다.

<Callout type="info" title="사은품 중복 적용 옵션">
- **우선 순위 적용**: 설정된 우선 순위가 높은 정책 하나만 적용합니다.
- **모든 정책 적용**: 조건에 맞는 모든 정책의 사은품을 지급합니다.
</Callout>

## 사은품 자동 적용 세부 정책 설정

### ① 사은품 세부 사항 입력

<AnnotatedImage src="gift-policy-detail-form" section="fulfillment" alt="사은품 세부 정책 설정 폼" caption="사은품 정책 세부 설정 화면" />

① **정책명** — 정책을 구분할 수 있는 이름을 입력합니다.

② **정책 적용 기준일** — [결제일] 또는 [주문일]을 선택하고 적용할 날짜 범위를 입력합니다.

③ **정책 적용 주문 타입** — [풀필먼트 주문에 적용] 또는 [판매자 출고 주문에 적용] 중 선택합니다.

④ **사은품 지급 수량 제한** — 원하는 제한 수량을 입력하거나, 제한이 없는 경우 [제한 없음]을 선택합니다.

<Callout type="warning">
제한 수량이 0이 되면 해당 정책의 사은품은 더 이상 지급되지 않습니다.
</Callout>

⑤ **판매 사이트 / 결제 금액 / 포함 상품** — 각 조건 토글을 활성화하여 사은품 지급 대상을 세부적으로 제한할 수 있습니다.

### ② 사은품 적용 대상 상품 설정

- **판매 사이트 상품코드** — 판매 사이트에 Qoo10 JP를 설정한 경우, 사이트의 상품코드를 기준으로 적용이 가능합니다. JQSM에서 확인한 상품코드를 입력합니다.

<AnnotatedImage src="gift-policy-qoo10-code" section="fulfillment" alt="Qoo10 JP 상품코드 확인" caption="JQSM 상품 목록 > 상품코드 확인" />

- **사은품 지급 대상 조건** — 설정한 상품의 주문 수량(상품별) 또는 한 장바구니의 전체 주문 수량을 기준으로 지급 조건을 설정합니다.
- **상품 타입** — 특정 상품 구매 시에만 사은품을 지급하려는 경우 적용합니다.

### ③ 사은품 설정

[Add SKU] 버튼을 클릭하여 사은품으로 지급할 상품의 SKU를 선택합니다.

### ④ 등록하기

모든 설정이 완료되면 [등록] 버튼을 클릭합니다. 설정한 정책 적용 기준일에 맞춰 사은품이 자동으로 적용됩니다.

## 예외 설정

### 수동 적용: 원하는 주문건에만 사은품 정책 적용

사은품 자동 적용 설정을 [수동 적용]으로 설정한 경우, 아래 절차로 특정 주문건에 정책을 수동으로 적용합니다.

<AnnotatedImage src="gift-policy-manual-apply" section="fulfillment" alt="사은품 정책 수동 적용" caption="주문 관리 > 사은품 관리 > 사은품 정책 수동 적용" />

1. **[주문 → 주문 관리]** 메뉴에 접속합니다.
2. 원하는 주문건을 선택합니다 (①).
3. **[사은품 관리 → 사은품 정책 수동 적용]** 을 선택합니다 (②).
4. 팝업에서 적용할 **사은품 정책**을 선택합니다 (③).
5. 추가 설정이 필요한 경우 입력합니다 (④).
6. **[사은품 정책 실행]** 버튼을 클릭합니다 (⑤).

### 정책 없이 사은품 직접 추가

사은품 정책 없이 특정 주문건에 사은품을 직접 추가하는 방법입니다.

<AnnotatedImage src="gift-policy-direct-add" section="fulfillment" alt="사은품 직접 추가" caption="주문 관리 > 사은품 관리 > 사은품 추가" />

1. **[주문 → 주문 관리]** 메뉴에 접속합니다.
2. 원하는 주문건을 선택합니다 (①).
3. **[사은품 관리 → 사은품 추가]** 를 선택합니다 (②).
4. **[사은품 직접 추가]** 를 선택합니다 (③).
5. 사은품을 추가한 후 **[사은품 추가]** 버튼을 클릭합니다 (④).

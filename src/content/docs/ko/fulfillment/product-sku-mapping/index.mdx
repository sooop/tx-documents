---

판매채널에서 단일 상품으로 전시·판매되지만, 실제 출고 시 2개 이상의 SKU를 조합해 발송해야 하는 상품이 있습니다. 스마트십의 **멀티옵션 세트(동적 세트)** 기능을 사용하면, 구매자가 선택한 옵션 조합에 따라 SKU들이 실시간으로 자동 구성되어 배송 주문에 등록됩니다.

미리 SET SKU를 구성하는 일반 세트 방식과 달리, 동적 세트는 수백억 가지 이상의 조합도 유연하게 처리할 수 있습니다.

## 적용 가능 사이트

- Qoo10 Japan
- 라쿠텐
- 쇼피
- 라자다
- 카페24
- 카카오 쇼핑

<Callout type="warning" title="Qoo10 전용 기능 안내">
현재 **동적 세트 SKU**는 Qoo10에 대해서만 적용 가능합니다.
</Callout>

## 이런 경우 사용을 추천합니다

- 색상·도수 등 속성이 많고 조합이 매우 다양한 상품을 세트로 판매하는 경우
- 수백 종의 SKU를 운영하며, 구매자의 선택에 따라 동적으로 세트를 구성해야 하는 경우
- 모든 조합마다 개별 SET SKU를 등록하기 어려운 경우
- 인기 SKU를 여러 상품의 공통 옵션으로 사용하는 경우

<Collapse title="사용 사례 예시">

**사례 1: 컬러 콘택트 렌즈 세트 (Qoo10)**

판매자가 22가지 색상 × 16단계 도수의 콘택트 렌즈를 판매하며, 구매자가 4종류의 렌즈(색상+도수)를 직접 선택해 세트를 구성합니다. 선택형 옵션으로 설정하면 약 360억 가지 조합이 발생하여 개별 SET SKU 등록이 불가능합니다. 동적 세트 기능을 사용하면 이를 자동으로 처리할 수 있습니다.

**사례 2: 침구 세트 (베개 + 커버)**

베개(소재·높이)와 커버(패턴·소재) 각 2개 옵션의 조합으로 세트를 구성하는 상품입니다. 각 SKU를 결정하는 옵션 수가 동일한 경우 동적 세트 기능으로 자동 맵핑이 가능합니다.

<AnnotatedImage src="sku-map-product-example" section="fulfillment" alt="침구 세트 상품 판매 예시" caption="Qoo10에서 베개+커버 조합 선택형 상품 예시 — 4개의 옵션 값이 2개 SKU를 각각 결정합니다." />

</Collapse>

## 설정 전 확인 사항

<Callout type="danger" title="옵션 수 일치 필수">
동적 세트에서 하나의 상품 내 각 SKU를 결정하기 위한 **옵션의 가짓수는 모든 구성 SKU에서 동일**해야 합니다.
</Callout>

- 옵션 값은 판매 사이트에 등록된 값과 **정확히 일치**해야 합니다.
- SKU 연결 정보에 미등록된 옵션이 주문에 포함되면 매칭에 실패하여 **판매자 배송 주문**으로 등록됩니다.

## 설정 방법

동적 세트 설정은 다음 세 단계로 진행합니다.

1. 판매채널에서 상품의 선택형 옵션 설정
2. 스마트십에서 멀티옵션 SKU 등록
3. 스마트십에서 세트 구성 정책 및 SKU 연결 정보 등록

---

### 1단계: 판매채널 선택형 옵션 설정 (판매 사이트)

판매 사이트(예: Qoo10 QSM)에서 상품의 옵션을 **선택형**으로 설정합니다.

**옵션 항목 순서 규칙**
- 각 상품을 구성하는 속성을 **SKU 단위로 순서대로** 작성합니다.
- 예: 베개(소재, 높이)와 커버(패턴, 소재) → `소재1`, `높이1`, `패턴2`, `소재2` 순서
- 옵션의 순서는 하나의 SKU를 결정하는 순서를 따라야 합니다.

| 올바른 순서 | 잘못된 순서 |
|------------|------------|
| 색1 → 도수1 → 색2 → 도수2 | 색1 → 색2 → 도수1 → 도수2 |

<Callout type="tip">
상품 선택과 무관한 확인용 옵션(예: 수령 확인)은 옵션 목록의 **맨 뒤**에 배치하세요.
</Callout>

<AnnotatedImage src="sku-map-qoo10-option-setting" section="fulfillment" alt="Qoo10 QSM 선택형 옵션 설정 화면" caption="Qoo10 QSM — 색1, 도수1, 색2, 도수2 순서로 선택형 옵션을 설정한 예시" />

---

### 2단계: SKU 등록 (스마트십)

**경로:** 상품 및 재고 > SKU 관리

스마트십에서 멀티옵션 상품의 각 옵션 조합에 해당하는 SKU를 등록합니다.

<AnnotatedImage src="sku-map-sku-type-multi" section="fulfillment" alt="SKU 등록 시 멀티옵션 선택" caption="대표 상품 등록 시 옵션유형을 '멀티옵션'으로 선택합니다." />

1. 상품 기본 정보를 입력하고 옵션유형을 **멀티옵션**으로 선택합니다.
2. 등록 후 표시되는 옵션 생성 화면에서 입력 방식을 **옵션 조합**으로 선택합니다.
3. 각 속성 이름과 속성값을 입력하고 **등록하기**를 클릭합니다.

<AnnotatedImage src="sku-map-sku-option-combination" section="fulfillment" alt="옵션 조합 입력 화면" caption="속성과 속성값을 입력하면 조합 가능한 모든 SKU가 자동 등록됩니다." />

<Callout type="warning">
조합의 수가 많은 경우 등록에 시간이 걸리며 화면이 멈춘 것처럼 보일 수 있습니다. 등록이 완료될 때까지 기다려 주세요.
</Callout>

---

### 3단계: 세트 구성 정책 등록

**경로:** 상품 및 재고 > 판매채널 상품-SKU 연결 > 멀티옵션 - 세트 연결

각 상품의 옵션 조합이 어떤 SKU에 대응하는지 정책으로 등록합니다.

#### 개별 등록 (직접 입력)

<AnnotatedImage src="sku-map-policy-setup" section="fulfillment" alt="세트 구성 정책 설정 화면" caption="세트 구성 정책 — 샵, 상품 번호, 조합할 옵션의 수를 설정합니다." />

1. **판매채널 Shop** — 정책을 적용할 샵을 선택합니다.
2. **지정 상품 번호** — `지정한 상품에 적용` 또는 `전체 상품에 적용` 중 선택합니다.
   - 여러 상품에 적용할 경우 상품 번호를 콤마(`,`)로 구분하여 입력합니다.
   - `전체 상품에 적용`은 샵당 1개만 설정 가능합니다.
3. **조합할 옵션의 수** — 하나의 SKU를 결정하는 데 필요한 옵션 수를 지정합니다.
4. **[등록하기]** 버튼을 클릭하여 정책을 등록합니다.

<Callout type="danger" title="전체 상품 적용 정책 주의">
전체 상품에 적용하는 정책을 선택하면 **판매자 상품코드/옵션코드로 SKU를 매칭하는 기능이 작동하지 않습니다.** 일부 특수한 판매점을 위한 옵션이므로 일반적인 경우에는 지정 상품 번호를 사용하세요.
</Callout>

정책 등록 후, 화면 하단의 SKU 연결 정보 패널에서 각 SKU와 옵션값을 등록합니다.

<AnnotatedImage src="sku-map-sku-link-register" section="fulfillment" alt="SKU 연결 정보 등록 화면" caption="SKU를 검색 또는 직접 입력하고, 해당 SKU에 매칭할 옵션값을 순서대로 입력합니다." />

1. **SKU no.**를 검색하거나 직접 입력합니다.
2. 정책에서 지정한 개수만큼 활성화된 **옵션값** 입력창에 값을 순서대로 입력합니다.
   - 옵션의 이름은 관계없으며, **옵션의 값**만 입력합니다.
   - 예: `チェック`, `綿100%`
3. **저장하기**를 클릭합니다.

#### 엑셀 일괄 등록

상단의 **[엑셀 업로드하기]** 버튼을 클릭하면 정책과 SKU 연결 정보를 한 번에 등록할 수 있습니다.

<AnnotatedImage src="sku-map-policy-setup-excel" section="fulfillment" alt="엑셀 업로드 화면" caption="엑셀 업로드 패널 — 샵, 옵션 항목 수, 세트의 상품 수, 판매채널 상품번호를 지정한 후 파일을 업로드합니다." />

1. **Site** — 상품을 판매하는 샵을 선택합니다.
2. **SKU에 매칭할 옵션 항목 수** — 하나의 SKU를 결정하는 옵션 수를 선택합니다. (예: 색상+도수 = 2)
3. **주문당 구성할 세트의 상품 수** — 세트를 이루는 SKU 수를 선택합니다. (예: 4종 상품 세트 = 4)
4. **판매채널 상품번호** — 대상 상품의 번호를 입력합니다.
5. SKU no.와 매칭 옵션값으로 구성된 엑셀 파일을 준비하여 업로드합니다.

<AnnotatedImage src="sku-map-policy-excel-data" section="fulfillment" alt="SKU 연결정보 엑셀 데이터 예시" caption="SKU 연결 정보 — 정책번호, 판매채널명, 상품번호, SKU no., 선택옵션값으로 구성됩니다." />

<Callout type="tip">
SKU 관리 페이지에서 SKU 정보를 엑셀로 다운로드한 후 편집하여 업로드하면 편리합니다.
</Callout>

---

## 주의사항

- 판매사이트 옵션값과 SKU 연결 정보의 옵션값 **순서 및 내용이 정확히 일치**해야 합니다.
- SKU 연결 정보의 (옵션 항목 수 × 상품 수)를 초과하는 옵션값은 무시됩니다.
- 연결 정보에 등록되지 않은 옵션이 주문에 포함되면 매칭 실패 → **판매자 배송 주문**으로 등록됩니다.

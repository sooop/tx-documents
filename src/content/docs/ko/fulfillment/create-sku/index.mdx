---
title: "SKU 생성하기"

---

풀필먼트 센터에 상품을 입고하려면 먼저 SKU를 생성해야 합니다.

<Callout type="info" title="메뉴 경로">
상품 및 재고 → [SKU 관리](https://smartship.tracxlogis.com/Inventory/SKUV2)
</Callout>

## SKU 타입

| 타입 | 설명 |
|------|------|
| **단일 상품** | 옵션이 없는 단일 상품에 사용 |
| **멀티 옵션** | 색상, 사이즈 등 다양한 옵션이 있는 상품에 사용 |

---

## 1. 개별 등록

### 1-1. 단일 상품 SKU 등록

<AnnotatedImage src="create-sku-single" section="fulfillment" alt="단일 상품 SKU 등록 화면" caption="SKU 관리 > 단일 상품 등록 화면" />

1. **[단일 상품]** 을 선택합니다.
2. **[상품명, 가격]** 을 입력한 후 **[중복 확인하기]** 를 선택합니다.
3. 해외 배송 상품의 경우 **[판매사이트 URL]** 을 입력합니다. URL만 입력하면 HS코드, 영문 수출명 등 수출에 필요한 기본 정보가 자동으로 입력됩니다.
4. **[등록하기]** 를 클릭합니다.

<Callout type="tip">
**[판매자 정의 코드]** 와 **[제품 바코드]** 를 함께 입력하면 재고 관리 및 API 연동 시 편리합니다.
</Callout>

<Callout type="info" title="최대 합포장 가능 수량">
부피가 큰 상품(예: 기저귀)의 경우 **[최대 합포장 가능 수량]** 을 설정하면, 설정 수량에 따라 포장이 자동 분리 처리됩니다.
</Callout>

### 1-2. 멀티 옵션 SKU 등록

<AnnotatedImage src="create-sku-multi" section="fulfillment" alt="멀티 옵션 SKU 등록 화면" caption="SKU 관리 > 멀티 옵션 등록 화면" />

1. **[멀티 옵션]** 을 선택합니다.
2. **[상품명, 가격]** 을 입력한 후 **[중복 확인하기]** 를 선택합니다.
3. 해외 배송 상품의 경우 **[판매사이트 URL]** 을 입력합니다.
4. **[신규 상품 등록 & 옵션 추가]** 를 선택합니다.
5. 옵션 생성기를 통해 옵션 조합을 입력합니다.
6. **[등록하기]** 를 클릭합니다.
7. 생성 결과를 확인합니다.

---

## 2. 엑셀 등록

여러 SKU를 한 번에 등록할 때 사용합니다.

<AnnotatedImage src="create-sku-excel" section="fulfillment" alt="SKU 엑셀 등록 화면" caption="SKU 관리 > 엑셀 업로드 화면" />

1. **[엑셀 업로드하기]** 를 선택합니다.
2. **[엑셀 포맷 다운로드하기]** 를 클릭하여 양식을 다운로드한 후 항목을 입력합니다.
3. **[파일 선택하기]** 를 통해 작성한 파일을 업로드합니다.
4. 업로드 결과를 확인합니다.
5. **[엑셀 등록하기]** 를 선택합니다.

입고 신청이 완료되면 카고 라벨을 출력한 후 화물을 센터로 발송합니다.

## 입고 가능 시간

- **평일(월~금):** 09:00 ~ 17:00
- **점심시간 제외:** 11:00 ~ 12:30
- **휴무:** 주말, 법정 공휴일, 대체 공휴일

## 입고 방식

입고 신청 완료 후 카고 라벨을 출력할 수 있습니다.

<Callout type="warning" title="카고 라벨 부착 필수">
- 출력한 카고 라벨을 입고 화물 또는 외부 박스에 **반드시 부착** 해 주세요.
- 카고 라벨 기준으로 입고 확인이 진행됩니다.
- 라벨 미부착 또는 정보 누락 시 입고가 지연될 수 있습니다.
</Callout>

## 카고 라벨 출력 방법

**경로:** SmartShip → 상품 및 재고 → 입고 신청 → 라벨 출력 → **카고 라벨 출력**

입고 신청 건을 선택한 후 **[라벨 출력]** 버튼을 클릭하거나, 마우스 오버 시 나타나는 메뉴에서 카고 라벨 출력을 선택합니다.

SmartShip에서 배송비를 결제하려면 TxMoney를 먼저 충전해야 합니다.

<Callout type="info">
### TxMoney란 무엇인가요?

- 스마트십 사용 시 발생하는 배송비와 기타 비용을 결제하기 위한 화폐입니다.
- 센터에 화물이 입고될 때, 해당 화물의 배송비가 결정되며 자동으로 사용자님의 TxMoney 계좌에서 차감됩니다.
- 이후 출고 후 발생하는 비용(풀필먼트 관련 부가 요금 및 관부가세 청구 등)은 청구 시점에 TxMoney에서 자동으로 차감됩니다.
- 입고 시 TxMoney 잔고가 부족한 경우, 입고에 실패하며 처리가 지연될 수 있으니, TxMoney는 충분한 잔고를 유지해주세요.
</Callout>

### 메뉴 위치

비용관리 >> TxMoney 충전

<AnnotatedImage src="topup-txmoney-page" section="settings" alt="TxMoney 충전 페이지 화면" caption="비용관리 > TxMoney 충전 화면" />

1. **나의 TxMoney 계좌** — 현재 보유 잔액과 미결제 청구 금액을 확인할 수 있습니다.
2. **충전 금액 입력** — 원하는 금액을 직접 입력하거나, 버튼을 눌러 빠르게 금액을 추가할 수 있습니다.
3. **무통장입금 충전** — 가상 계좌를 발급받아 입금하는 방식입니다.
4. **Paynow / 신용/직불 카드결제** — 카드로 즉시 결제하는 방식입니다.

### 충전방법

원하시는 충전 금액을 선택한 후, 아래 결제 방식 중 하나를 선택해주세요.

<Collapse title="무통장입금 (가상 계좌)">

1. **[가상 계좌로 입금]** 버튼을 클릭합니다.
2. 표기된 가상 계좌 번호로 입금합니다.
   - 예금주명: **트랙스_회사명** (예: 트랙스_홍길동무역)
3. 입금 후 자동으로 충전이 완료됩니다.

<Callout type="tip">
가상계좌는 발급일로부터 **7일간만 유효**합니다. 만료된 경우 가상 계좌 입금을 다시 신청해야 합니다.
</Callout>

</Collapse>

<Collapse title="국민·BC·우리·롯데카드 결제 (국내 결제 · 수수료 없음)">

1. **[국민·BC·우리·롯데카드]** 버튼을 클릭합니다.
2. 약관에 동의한 후 카드사를 선택합니다.
3. **[결제]** 버튼을 클릭하여 결제를 진행합니다.
4. 결제 완료 후 자동으로 충전이 완료됩니다.

</Collapse>

<Collapse title="모든 카드 결제 (해외 결제 · Stripe PG)">

1. **[모든 카드 결제]** 버튼을 클릭합니다.
2. 주의 사항을 확인 후 **[결제하기]** 버튼을 클릭합니다.
3. 카드번호, 유효기간, CVC 코드를 입력합니다.
4. 카드 소유자 이름을 입력합니다.
5. **[결제]** 버튼을 클릭합니다.
6. 결제 완료 후 자동으로 충전이 완료됩니다.

</Collapse>

<AnnotatedImage src="stripe-pg-page" section="settings" alt="Stripe 카드 결제 화면" caption="Paynow / 신용·직불 카드결제 — 카드 정보 입력 화면" />

<Callout type="warning">
### 카드 결제 유의 사항

- [모든 카드 결제]는 싱가포르의 'Stripe' PG를 사용하고 있으며, 카드 결제 내역에는 승인 국가가 'SINGAPORE'로 표시됩니다.
- 싱가포르 PG를 사용하기 때문에 해외 원화 결제(KRW)로 승인됩니다.
- 만약, 사용중인 카드사에서 **해외 원화 결제 차단 서비스**를 설정해 두셨다면, 결제가 정상적으로 이루어지지 않을 수 있습니다.
- TxMoney 충전 시 통화가 SGD가 아닌 경우, 카드사의 정책에 따라 **해외 거래 수수료가 부과될 수 있습니다.**
</Callout>

## 충전 및 사용 안내
- 판매자 출고 이용 시, 센터에 화물이 도착하면 배송비가 자동으로 계산되어 TxMoney 잔고에서 차감됩니다.
- TxMoney를 충전한 후 일부 금액을 사용한 후에는 환불이 불가합니다. 
- 스마트십 서비스 탈퇴 시에는 보유 잔고의 5% 공제 후 환불 받을 수 있습니다. (신청일 기준 익월 15일 환불 진행)
- 결제를 완료하였으나 TxMoney가 충전되지 않을 경우 또는 오충전의 경우 사용 전이면 취소 요청이 가능합니다. 아래 고객센터 또는 메일로 문의 부탁 드립니다.

   - 일본 사업자 : help-jp@tracxlogis.com 또는 채팅상담

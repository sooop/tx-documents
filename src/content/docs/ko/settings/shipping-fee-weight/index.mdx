## 무게 측정 기준

SmartShip은 배송비 산정 시 아래 두 가지 무게를 비교하여 더 큰 값을 적용합니다.

| 구분 | 설명 |
|------|------|
| **실중량** | 포장 완료 후 박스 전체의 실제 무게 (단위: kg) |
| **부피무게** | 가로(cm) × 세로(cm) × 높이(cm) ÷ 6,000 (단위: kg) |
| **적용 무게** | 실중량과 부피무게 중 **더 큰 값** |

<Callout type="info" title="부피무게란?">
부피가 크지만 가벼운 상품(예: 스티로폼, 완충재가 많은 포장 등)은 실중량보다 부피무게가 더 크게 산출될 수 있습니다. 이 경우 부피무게를 기준으로 배송비가 책정됩니다.
</Callout>

## 사이즈 측정 방법

박스의 **외부 최대 치수**를 기준으로 가로·세로·높이를 측정합니다.

<div style={{display: "flex", justifyContent: "center", margin: "2rem 0"}}>
  <img src="/images/settings/box-dimension-guide.svg" alt="박스 치수 측정 가이드 — 가로·세로·높이" width="460" height="325" />
</div>

| 치수 | 측정 기준 |
|------|----------|
| **가로** | 박스 정면의 좌우 길이 |
| **세로** | 박스의 앞뒤 깊이 |
| **높이** | 박스의 위아래 높이 |

<Callout type="warning" title="측정 시 주의사항">
- 박스 **외부**의 최대 치수를 기준으로 측정합니다.
- 테이프, 손잡이 등 돌출 부분이 있는 경우 해당 부분을 **포함하여** 측정합니다.
- 변형되거나 찌그러진 경우에도 최대 외부 치수를 기준으로 합니다.
</Callout>

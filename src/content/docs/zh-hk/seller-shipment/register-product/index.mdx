在 SmartShip 中註冊訂單之前，需要先註冊待發貨的商品。系統支援**單件註冊**和 **Excel 批量上傳**兩種方式。

<Callout type="info" title="商品註冊入口">
在頂部選單進入**商品管理**，點擊 **+ 註冊** 按鈕即可開啟單件註冊頁面。批量註冊請使用同一頁面的 **Excel 上傳** 按鈕。
</Callout>

## 單件商品註冊

商品註冊頁面由**商品資訊**、**海外配送清關資訊**、**銷售渠道商品資訊**三個標籤頁組成。按順序填寫後點擊**註冊**按鈕即可。

### 第一步：商品資訊

在**商品資訊**標籤頁輸入商品基本資訊。

<AnnotatedImage src="product-reg-basic-info" section="seller-shipment" alt="商品資訊標籤頁 — 基本資訊輸入頁面" caption="商品資訊標籤頁 — 商品基本資訊" />

| 欄位 | 必填 | 說明 |
|------|:----:|------|
| TX 商品編碼 | — | 系統自動分配，不可修改 |
| 賣家自訂編碼 | | 賣家自行設定的內部管理編碼 |
| 商品條碼 | | 輸入條碼後點擊**確認**，可自動取得已有商品資訊 |
| 商品名稱 | ✓ | 至少輸入 3 個字元 |
| 選項名稱 | | 如有顏色、尺寸等選項，在此輸入 |
| 價格 / 貨幣 | | 銷售單價及貨幣單位（預設：KRW） |
| 重量（kg） | | 輸入 0.1 以上的數值 |
| 尺寸（cm × cm × cm） | | 寬 × 長 × 高 |
| 原產地 | | 從下拉選單中選擇 |
| 材質 | | 選擇材質類型或手動輸入 |
| 含電池 | | 選擇是否含電池（預設：不含） |
| 敏感商品 | | 選擇適用 / 不適用 |
| HS 編碼 | | 手動輸入關稅分類編碼 |

### 第二步：海外配送清關資訊

在**海外配送清關資訊**標籤頁填寫出口清關所需資訊。

<AnnotatedImage src="product-reg-customs-info" section="seller-shipment" alt="海外配送清關資訊標籤頁輸入頁面" caption="海外配送清關資訊標籤頁 — 商品清關資訊管理" />

| 欄位 | 說明 |
|------|------|
| 清關商品名稱 | 出口申報時使用的英文品名，必須以**英文大寫**輸入 |
| 商品條碼 | 條碼編號 |
| 銷售網站 URL | 商品所在銷售頁面的 URL |
| HS / HSN | 按大類 → 中類 → 小類依次選擇，或直接輸入編碼 |
| 材質 | 材質種類及成分比例 |
| 含電池 | 是否含電池 |
| 敏感商品 | 是否屬於敏感品目 |
| 型號 / 品牌 | 產品型號及品牌名稱 |
| 原產地 / 製造商 | 生產國及製造商名稱 |
| 清關商品名稱（zh-cn） | 發往中國時填寫中文品名 |
| 商品稅號 | 海關申報用稅則號 |
| GST 稅率（%） | 發往 GST 適用國家（如澳洲）時填寫（預設：0） |
| VAT ID | VAT 登記編號 |
| EORI | 發往歐洲時填寫 EORI 編號 |

<Callout type="warning" title="清關商品名稱注意事項">
清關商品名稱必須以**英文大寫**填寫。使用中文、韓文或特殊符號可能導致清關錯誤。
</Callout>

### 第三步：銷售渠道商品資訊

在**銷售渠道商品資訊**標籤頁中，可查看並設定各已串接銷售平台的商品資訊。

填寫完所有資訊後，點擊**註冊**按鈕即可完成商品註冊。

## 批量商品註冊（Excel 上傳）

需要一次性註冊多件商品時，請使用 Excel 上傳功能。

<AnnotatedImage src="product-bulk-reg" section="seller-shipment" alt="批量配送商品註冊 — Excel 上傳頁面" caption="批量配送商品註冊頁面" />

### 下載 Excel 範本

1. 點擊批量配送商品註冊頁面右上角的 **下載 XLSX 格式** 按鈕。
2. 開啟下載的 Excel 檔案，填寫商品資訊。

<Callout type="warning" title="新版範本說明（自 2024 年 7 月 11 日起適用）">
新版範本的欄位名稱中包含編碼和括號，請**勿刪除**這些編碼及括號，否則上傳將無法正常處理。
</Callout>

### 欄位說明

<Collapse title="查看全部欄位說明">

| 欄位名稱 | 編碼 | 說明 |
|----------|------|------|
| 商品名稱 | M00 | 商品名稱（必填） |
| 發貨數量 | M01 | 每次發貨的數量 |
| 單價 | M02 | 商品單價 |
| 貨幣 | M03 | 貨幣代碼（如：KRW、USD、JPY） |
| 細分類別 | M04 | 商品細分類別 |
| 選項 | M05 | 選項資訊（顏色、尺寸等） |
| 商品 URL | M06 | 銷售頁面 URL |
| 商品材質（成分） | M07 | 材質及成分資訊 |
| HS/JAL | M08 | HS 編碼或 JAL 編碼 |
| Overseas / Trade code | — | 海外貿易編碼（HS Code、JAL 等） |
| 品牌 | M10 | 品牌名稱 |
| Shop Name | M11 | 銷售網站名稱 |
| 條碼 | M12 | 商品條碼 |
| 行郵稅 | M13 | 是否適用行郵稅 |
| 行郵稅編碼（後半部分） | M14 | 行郵稅編碼末尾部分 |
| 賣家自訂商品編碼 | M15 | 賣家內部管理編碼 |
| GST Rate | M16 | GST 稅率（%） |

</Collapse>

### 上傳檔案

1. 在下拉選單中選擇**註冊**。
2. 點擊**選擇檔案**按鈕，選擇已填寫好的 Excel 檔案。
3. 上傳後，透過**處理結果（ST0）**和**詳細資訊（ST1）**欄確認註冊結果。

<Callout type="tip" title="上傳失敗時">
若處理結果（ST0）顯示錯誤，請查看詳細資訊（ST1）了解原因，修正相應條目後重新上傳。
</Callout>

## 查看和管理已註冊商品

商品註冊完成後，在**商品管理**選單的商品查詢中查看列表。

<AnnotatedImage src="product-management" section="seller-shipment" alt="商品管理 — 商品查詢頁面" caption="商品管理 — 商品查詢列表" />

- **關鍵字搜尋**：按商品編碼、商品名稱等條件篩選
- **下載查詢內容**：將目前查詢結果匯出為 Excel 檔案
- **Excel 上傳**：用於批量修改已註冊的商品資訊

<Callout type="tip" title="批量修改商品資訊">
如需修改已註冊商品的資訊，請點擊 **Excel 上傳**，在下拉選單中選擇**修改**，點擊**下載修改格式**按鈕取得目前商品資料，編輯完成後重新上傳。
</Callout>

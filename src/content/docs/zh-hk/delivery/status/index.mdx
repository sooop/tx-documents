在配送狀態選單中，您可以一目了然地查看所有已發貨訂單的配送狀態，並逐筆查詢追蹤資訊和運費明細。

<Callout type="info" title="選單路徑">
**配送管理 → 配送狀態**
</Callout>

## 查詢配送訂單

<AnnotatedImage src="delivery-status-page" section="delivery" alt="配送狀態完整頁面" caption="配送管理 > 配送狀態頁面" />

### 設定搜尋條件

<AnnotatedImage src="delivery-status-search" section="delivery" alt="配送訂單搜尋篩選條件" caption="配送訂單查詢 — 搜尋條件" />

| 搜尋條件 | 說明 |
|----------|------|
| 基準日期 | 選擇訂單登記日、出庫日等日期基準後指定查詢區間（可使用 1W / 1M 按鈕快速選擇） |
| 配送訂單類型 | 依全部訂單或特定訂單類型篩選 |
| 出貨倉庫 | 依出貨倉庫篩選 |
| 配送區間 | 選擇出發國 → 到達國 |
| 銷售網站 | 依已連接的銷售渠道篩選 |
| 訂單登記方式 | 依全部訂單或登記方式篩選 |
| 其他條件 | 單獨篩選預發號後運單號已變更的訂單，或超過 150 美元的訂單 |
| 關鍵字搜尋 | 依 QSP NO.、訂單號、收件人姓名等搜尋 |

### 發貨狀態摘要

<AnnotatedImage src="delivery-status-summary" section="delivery" alt="發貨狀態摘要面板" caption="發貨狀態 — 各狀態訂單數量摘要" />

頁面頂部的摘要面板可讓您一目了然地查看各狀態的訂單數量。

| 狀態 | 說明 |
|------|------|
| 全部配送開始 | 已開始配送的總訂單數 |
| 配送中 | 目前正在配送中的訂單數 |
| 配送完成 | 最終投遞已完成的訂單數 |
| 配送失敗 | 已處理為投遞失敗的訂單數 |
| 退件中 | 正在退件中的訂單數 |
| 退件完成 | 退件已完成的訂單數 |

點擊任意狀態項目，即可直接篩選該狀態的訂單列表。點擊右上角的**查看每日發貨件數**按鈕，可查看按日期統計的發貨資料。

### 訂單列表

<AnnotatedImage src="delivery-status-list" section="delivery" alt="配送狀態訂單列表" caption="配送狀態 — 訂單列表" />

在查詢結果列表中，可查看 QSP NO.、運單號、配送狀態、出發國/到達國等主要資訊。點擊列表底部的 **Excel** 按鈕，可將查詢內容下載為 Excel 檔案。

## 查看訂單詳細資訊

點擊列表中的訂單，底部將展開詳細資訊面板。面板由**配送追蹤資訊**、**運費資訊**、**商品資訊**三個標籤頁組成。

### 配送追蹤資訊

<AnnotatedImage src="delivery-status-tracking" section="delivery" alt="配送追蹤資訊標籤" caption="配送追蹤資訊 — 各環節處理記錄" />

可依時間順序查看從接收配送訂單到最終投遞的各環節處理記錄。

| 欄位 | 說明 |
|------|------|
| 狀態 | 處理階段名稱（配送訂單接收、集貨中心入庫完成、出庫完成、到達國清關開始/完成、移交配送商等） |
| 日期 | 狀態變更的日期和時間 |
| 說明 | 附加備註（如預計出發機場等） |
| 配送員 | 當地配送員資訊（如有） |

點擊**配送詳細資訊**按鈕，可前往配送商的原始追蹤頁面。

### 運費資訊

<AnnotatedImage src="delivery-status-fee" section="delivery" alt="運費資訊標籤" caption="運費資訊 — 各項費用明細" />

可查看該訂單收取的運費各項明細。

| 欄位 | 說明 |
|------|------|
| 運費項目 | 項目名稱（基本運費、燃油附加費、EDI 傳輸費等） |
| 原始貨幣 | 計費貨幣單位 |
| 原始貨幣金額 | 各項收費金額 |

列表最底部的 **Total Fee** 列顯示運費總計。

SmartShipで注文を登録するには、まず発送する商品を登録する必要があります。商品の登録方法は**個別登録**と**一括Excelアップロード**の2種類に対応しています。

<Callout type="info" title="商品登録メニューの場所">
上部メニューの**商品管理**に移動し、**+ 登録**ボタンをクリックすると個別登録画面が開きます。一括登録は同じページの**Excelアップロード**ボタンをご利用ください。
</Callout>

## 個別商品登録

商品登録画面は**商品情報**、**海外配送通関情報**、**販売チャンネル商品情報**の3つのタブで構成されています。タブの順番通りに入力した後、**登録**ボタンをクリックします。

### ステップ1：商品情報

**商品情報**タブでアイテムの基本情報を入力します。

<AnnotatedImage src="product-reg-basic-info" section="seller-shipment" alt="商品情報タブ — アイテム基本情報入力画面" caption="商品情報タブ — アイテム基本情報" />

| 項目 | 必須 | 説明 |
|------|:----:|------|
| TX 商品コード | — | システムが自動付与。変更不可 |
| 販売者定義コード | | 販売者が独自に設定する社内管理コード |
| 商品バーコード | | バーコード番号を入力し**確認**をクリックすると既存商品情報を自動取得 |
| 商品名 | ✓ | 3文字以上で入力 |
| オプション名 | | カラー・サイズなどオプション区分が必要な場合に入力 |
| 価格 / 通貨 | | 販売単価と通貨単位を選択（デフォルト：KRW） |
| 重量（kg） | | 0.1以上の数値を入力 |
| サイズ（cm × cm × cm） | | 幅 × 長さ × 高さ |
| 原産国 | | ドロップダウンから選択 |
| 素材 | | 素材の種類を選択または直接入力 |
| バッテリー内蔵 | | バッテリーの有無を選択（デフォルト：含まない） |
| センシティブ商品 | | 該当なし / 該当を選択 |
| HSコード | | 関税分類コードを直接入力 |

### ステップ2：海外配送通関情報

**海外配送通関情報**タブで輸出通関に必要な情報を入力します。

<AnnotatedImage src="product-reg-customs-info" section="seller-shipment" alt="海外配送通関情報タブ入力画面" caption="海外配送通関情報タブ — 商品通関情報管理" />

| 項目 | 説明 |
|------|------|
| 通関商品名 | 輸出申告時に使用する英文品目名。**英大文字**で入力すること |
| 商品バーコード | バーコード番号 |
| 販売サイトURL | 商品が登録されている販売ページのURL |
| HS / HSN | 大分類 → 中分類 → 小分類の順に選択するか、コードを直接入力 |
| 素材 | 素材の種類および成分比率 |
| バッテリー内蔵 | バッテリーの有無 |
| センシティブ商品 | センシティブ品目への該当有無 |
| モデル名 / ブランド | 製品モデル名およびブランド |
| 原産国 / メーカー | 生産国およびメーカー名 |
| 通関商品名（zh-cn） | 中国向け発送時の中国語品目名 |
| 製品税番 | 税関申告用の税番コード |
| GST税率（%） | GST適用国（オーストラリアなど）向け発送時に入力（デフォルト：0） |
| VAT ID | VAT登録番号 |
| EORI | ヨーロッパ向け発送時のEORI番号 |

<Callout type="warning" title="通関商品名に関する注意">
通関商品名は必ず**英大文字**で入力してください。日本語・ハングル・特殊文字を使用すると通関エラーが発生する場合があります。
</Callout>

### ステップ3：販売チャンネル商品情報

**販売チャンネル商品情報**タブでは、連携済みのショッピングモールごとの商品情報を確認し、マッピングすることができます。

すべての情報を入力後、**登録**ボタンをクリックすると商品登録が完了します。

## 一括商品登録（Excelアップロード）

複数の商品を一度に登録する場合は、Excelアップロード機能を使用します。

<AnnotatedImage src="product-bulk-reg" section="seller-shipment" alt="一括配送商品登録 — Excelアップロード画面" caption="一括配送商品登録画面" />

### Excelフォームのダウンロード

1. 一括配送商品登録画面の右上にある**XLSXフォーマットをダウンロード**ボタンをクリックします。
2. ダウンロードしたExcelファイルを開き、商品情報を入力します。

<Callout type="warning" title="新フォーマットのご案内（2024年7月11日より適用）">
新フォーマットのカラム名にはコードと括弧が含まれています。該当コードおよび括弧を**削除しないでください**。削除するとアップロードが正常に処理されません。
</Callout>

### カラムの説明

<Collapse title="全カラムの説明を見る">

| カラム名 | コード | 説明 |
|----------|--------|------|
| 商品名 | M00 | 商品名（必須） |
| 発送数量 | M01 | 発送単位数量 |
| 単価 | M02 | 商品単価 |
| 通貨 | M03 | 通貨コード（例：KRW、USD、JPY） |
| 細部分類 | M04 | 商品の細部分類 |
| オプション | M05 | オプション情報（カラー、サイズなど） |
| 商品URL | M06 | 販売ページURL |
| 商品素材（成分） | M07 | 素材および成分情報 |
| HS/JAL | M08 | HSコードまたはJALコード |
| Overseas / Trade code | — | 海外取引コード（HSコード、JALなど） |
| ブランド | M10 | ブランド名 |
| Shop Name | M11 | 販売サイト名 |
| バーコード | M12 | 商品バーコード番号 |
| 行郵税 | M13 | 行郵税適用の有無 |
| 行郵税コード（後半部分） | M14 | 行郵税コードの末尾部分 |
| ユーザー定義商品コード | M15 | 販売者の社内管理コード |
| GST Rate | M16 | GST税率（%） |

</Collapse>

### ファイルのアップロード

1. ドロップダウンから**登録**を選択します。
2. **ファイルを選択**ボタンをクリックし、作成したExcelファイルを選択します。
3. アップロード後、**処理結果（ST0）**と**詳細内訳（ST1）**のカラムで登録結果を確認します。

<Callout type="tip" title="アップロードに失敗した場合">
処理結果（ST0）にエラーが表示された場合は、詳細内訳（ST1）で原因を確認し、該当項目を修正して再アップロードしてください。
</Callout>

## 登録済み商品の確認・管理

商品登録後は、**商品管理**メニューの商品照会で一覧を確認します。

<AnnotatedImage src="product-management" section="seller-shipment" alt="商品管理 — 商品照会画面" caption="商品管理 — 商品照会リスト" />

- **キーワード検索**：商品コード・商品名などの条件でフィルタリング
- **照会内容のダウンロード**：現在の照会結果をExcelファイルとしてエクスポート
- **Excelアップロード**：登録済み商品情報を一括修正する際に使用

<Callout type="tip" title="商品情報の一括修正">
登録済み商品の情報を修正するには、**Excelアップロード**をクリックしてドロップダウンから**修正**を選択し、**修正用フォーマットのダウンロード**ボタンで現在の商品データを取得して編集後、再アップロードしてください。
</Callout>

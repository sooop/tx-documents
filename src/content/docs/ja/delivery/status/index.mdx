配送状況メニューでは、発送済み注文の配送状態を一覧で確認し、件ごとに追跡情報と送料明細を照会できます。

<Callout type="info" title="メニューパス">
**配送管理 → 配送状況**
</Callout>

## 配送注文を照会する

<AnnotatedImage src="delivery-status-page" section="delivery" alt="配送状況ページ全体" caption="配送管理 > 配送状況画面" />

### 検索条件の設定

<AnnotatedImage src="delivery-status-search" section="delivery" alt="配送注文の検索フィルター" caption="配送注文履歴照会 — 検索条件" />

| 検索条件 | 説明 |
|----------|------|
| 基準日の選択 | 注文登録日・出荷日などの日付基準を選択し、照会期間を指定（1W / 1M ボタンで素早く選択） |
| 配送注文タイプ | 全注文または特定の注文タイプでフィルタリング |
| 出荷倉庫 | 出荷倉庫を基準にフィルタリング |
| 配送区間 | 出発国 → 到着国を選択 |
| 販売サイト | 連携済みの販売チャネル別にフィルタリング |
| 注文登録方式 | 全注文または登録方式別にフィルタリング |
| その他の条件 | 事前発番後に送り状番号が変更された注文、150ドル超の注文を別途フィルタリング |
| キーワード検索 | QSP NO.、注文番号、受取人名などで検索 |

### 発送状況サマリー

<AnnotatedImage src="delivery-status-summary" section="delivery" alt="発送状況サマリーパネル" caption="発送状況 — ステータス別件数" />

ページ上部のサマリーパネルで、ステータス別の件数を一目で確認できます。

| ステータス | 説明 |
|-----------|------|
| 配送開始合計 | 配送が開始された全件数 |
| 配送中 | 現在配送中の件数 |
| 配送完了 | 最終配達が完了した件数 |
| 配送失敗 | 配達失敗として処理された件数 |
| 返送中 | 返送が進行中の件数 |
| 返送完了 | 返送が完了した件数 |

各ステータス項目をクリックすると、該当ステータスの注文リストに絞り込まれます。右上の**日別発送件数を確認する**ボタンをクリックすると、日別の発送統計を確認できます。

### 注文リスト

<AnnotatedImage src="delivery-status-list" section="delivery" alt="配送状況注文リストテーブル" caption="配送状況 — 注文リスト" />

照会された注文リストでは、QSP NO.、送り状番号、配送状態、出発・到着国などの主要情報を確認できます。リスト下部の **Excel** ボタンをクリックすると、照会内容をExcelファイルでダウンロードできます。

---

## 件別詳細情報の確認

リストの注文をクリックすると、下部に詳細情報パネルが開きます。**配送追跡情報**、**送料情報**、**商品情報**の3つのタブで構成されています。

### 配送追跡情報

<AnnotatedImage src="delivery-status-tracking" section="delivery" alt="配送追跡情報タブ" caption="配送追跡情報 — ステップ別処理履歴" />

注文受付から最終配達まで、ステップ別の処理履歴を時系列で確認できます。

| カラム | 説明 |
|--------|------|
| ステータス | 処理ステージ名（配送注文受付、集荷センター入庫完了、出庫完了、到着国通関開始/完了、配送業者引き渡しなど） |
| 日付 | そのステータスに変更された日時 |
| 説明 | 追加メモ（出発予定空港など） |
| 配送担当者 | 現地配送担当者の情報（該当する場合） |

**配送詳細情報**ボタンをクリックすると、配送業者の追跡ページに移動します。

### 送料情報

<AnnotatedImage src="delivery-status-fee" section="delivery" alt="送料情報タブ" caption="送料情報 — 項目別費用明細" />

該当注文に請求された送料の項目別明細を確認できます。

| カラム | 説明 |
|--------|------|
| 送料項目 | 基本送料、燃油サーチャージ、EDI転送料などの項目名 |
| 原通貨 | 請求通貨単位 |
| 原通貨金額 | 項目別の請求金額 |

リスト最下部の **Total Fee** 行で送料の合計を確認できます。

SmartShipで送料を支払うには、まずTxMoneyをチャージする必要があります。

<Callout type="info">
### TxMoneyとは？

- SmartShip利用時に発生する送料やその他費用を支払うための通貨です。
- センターに荷物が入庫されると、その荷物の送料が確定し、自動的にTxMoneyアカウントから差し引かれます。
- 出庫後に発生する費用（フルフィルメント関連の追加料金や関税・消費税の請求など）も、請求時にTxMoneyから自動的に差し引かれます。
- 入庫時にTxMoneyの残高が不足している場合、入庫に失敗し処理が遅延する場合があります。常に十分な残高を維持してください。
</Callout>

### メニューの場所

コスト管理 >> TxMoneyチャージ

<AnnotatedImage src="topup-txmoney-page" section="settings" alt="TxMoneyチャージページ" caption="コスト管理 > TxMoneyチャージ画面" />

1. **マイTxMoneyアカウント** — 現在の残高および未払い請求金額を確認できます。
2. **チャージ金額の入力** — 金額を直接入力するか、クイック追加ボタンで金額を選択します。
3. **銀行振込チャージ** — バーチャル口座番号を発行して振り込む方式です。
4. **Paynow / クレジット・デビットカード決済** — カードで即時決済します。

### チャージ方法

1. チャージしたい金額を選択または入力してください。
2. お支払い方法を選択してください。

* 銀行振込（バーチャル口座）
    - **[銀行振込チャージ]** ボタンをクリック
    - 表示されたバーチャル口座番号に振り込みます（口座名義：**Tracx\_会社名**、例：Tracx\_山田商事）
    - 入金確認後、自動的にチャージ完了
      バーチャル口座は7日間のみ有効です。期限切れの場合は再度申請してください。
* 国民・BC・ウリ・ロッテカード（手数料無料）— 韓国国内決済
    1. **[国民・BC・ウリ・ロッテカード]** ボタンをクリック
    2. 利用規約に同意後、カード会社を選択
    3. **[決済]** をクリックして決済を進める
    4. 決済後、自動的にチャージ完了
* すべてのカード（海外決済）
    1. **[すべてのカード]** ボタンをクリック
    2. 注意事項を確認して **決済する** をクリック
    3. カード番号、有効期限、CVCを入力
    4. カード名義人の名前を入力
    5. **[決済]** ボタンをクリック
    6. 決済完了後、自動的にチャージ完了

<AnnotatedImage src="stripe-pg-page" section="settings" alt="Stripeカード決済画面" caption="Paynow / クレジット・デビットカード — カード情報入力画面" />

<Callout type="warning">
### カード決済に関する注意事項

- **[すべてのカード]** はシンガポールのStripe決済代行を使用しており、カード明細には承認国が **SINGAPORE** と表示されます。
- シンガポールのPGを使用するため、海外KRW建て決済として処理されます。
- ご利用のカード会社で**海外KRW決済ブロックサービス**を設定している場合、決済が正常に行われない場合があります。
- チャージ時の通貨がSGD以外の場合、カード会社の規定により**海外取引手数料が発生する場合があります。**
</Callout>

## チャージ・ご利用に関するご案内

- 販売者出庫ご利用の場合、センターに荷物が到着すると送料が自動的に計算され、TxMoney残高から差し引かれます。
- TxMoneyをチャージ後、一部ご利用いただいた場合、残高の返金はできません。
- SmartShipを退会される場合、保有残高の5%を差し引いた金額が返金されます。（申請日の翌月15日に返金処理）
- 決済は完了したがTxMoneyにチャージされなかった場合、または誤チャージの場合は、ご利用前であればキャンセル申請が可能です。下記サポートまでお問い合わせください。
   - 韓国事業者：help-kr@tracxlogis.com またはチャットサポート
   - 日本事業者：help-jp@tracxlogis.com またはチャットサポート
   - シンガポール事業者：help-sg@tracxlogis.com
